Quick Reference

  • A vector is drawn as an arrow: length = magnitude, arrowhead = direction.
  • Symbol: ⃗A = vector (has direction); A = scalar magnitude only.
  • Scalar mult.: positive scalar → same direction; negative scalar → flips direction.
  • Vector addition: use tip-to-tail. Resultant = tail of first to tip of last.
  • Perpendicular vectors: A² + B² = C²  |  Collinear: add or subtract magnitudes.

Answer Key

Part A — Drawing Vectors

Problems 1–4: Arrows should point in the correct direction (right, down, up, 45° up-right) with the magnitude labeled above each arrow. Longer magnitude = longer arrow.
Problem 5 (2⃗A): Arrow pointing right, twice as long as ⃗A (12 m). Original ⃗A = 6 m right.
Problem 6 (−⃗A): Arrow pointing left with magnitude 6 m. Negative flips the direction.

Part B — Labeling

Problem 7
⃗F = 20 N East; F = 20 N
Arrow over letter = vector; plain letter = scalar
Problem 8
⃗D = 15 km South; D = 15 km
Direction noted only with the vector symbol
Problem 9 — Scalar Multiplication (⃗A = 8 m right):
⃗A: magnitude 8 m, direction right  |  2⃗A: magnitude 16 m, direction right  |  0.5⃗A: magnitude 4 m, direction right  |  −2⃗A: magnitude 16 m, direction left (opposite)

Part D — Commutative Property

Problem 14: Both orders give the same resultant: C = √(6² + 8²) = √100 = 10 m. This demonstrates the commutative property of vector addition (⃗A + ⃗B = ⃗B + ⃗A).
