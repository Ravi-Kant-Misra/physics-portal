Drawing Vectors

➡
What is a Vector Arrow?

A vector is a quantity that has both magnitude (size) and direction. Vectors are drawn as arrows:

Magnitude
→ length
The length of the arrow shows how big the vector is. Double magnitude = double arrow length.
Direction
➡ arrowhead
The arrowhead points in the direction of the vector.
How to draw: Point the arrow in the vector's direction, then write the magnitude above the arrow. Bigger magnitude = longer arrow.
Four vector examples — each arrow shows direction; length proportional to magnitude
Compass map — vectors point in cardinal directions

Vector Symbol vs. Scalar Symbol

A vector is written as a letter with an arrow above it. The same letter without an arrow is just the scalar magnitude (no direction).

Vector (has direction)
⃗A
Arrow overhead = full vector (magnitude + direction)
Scalar magnitude (no direction)
A
Plain letter = just the size (a positive number)
Three labeled vectors — arrow above letter signals direction is included
×
Scalar Multiplication of Vectors

Multiplying a vector by a scalar (a number) changes its magnitude. The direction rules are important:

Positive scalar × vector → multiply magnitude, keep same direction
Negative scalar × vector → multiply magnitude, FLIP direction (180°)
Scalar multiplication — negative scalar flips the vector's direction
⚠ Key rule: Negative scalar does NOT mean the vector points down — it means the vector points opposite to its original direction.
➕

Adding Vectors — 3 Steps

1
Connect tip-to-tail Place the tail (start) of the second vector at the tip (end) of the first vector. This is the tip-to-tail method. The order does not matter for the final result.
2
Draw the resultant Draw a new vector from the tail of the FIRST vector to the tip of the LAST vector. This is the resultant (total vector).
3
Find the magnitude with Pythagorean theorem For perpendicular vectors: A² + B² = C². For collinear vectors (same/opposite direction): simply add or subtract.
✓ Correct: Tip-to-Tail
Tail of B starts where tip of A ends
✘ Incorrect: Tails Together
Both tails at same point — this is wrong for addition
△
Example: Perpendicular Vectors (Pythagorean Theorem)
Tip-to-tail: 4 right + 3 up = resultant C

Calculation

A² + B² = C²
4² + 3² = C²
16 + 9 = C²
25 = C²
C = 5 units
The resultant goes from the tail of the first vector to the tip of the last vector.
↔
Collinear Vectors (Same Line)

When vectors point in the same direction or opposite directions, simply add or subtract their magnitudes.

Collinear vectors — same direction: add | opposite direction: subtract
⇌
Commutative Property of Vector Addition
Order doesn't matter!  ⃗A + ⃗B = ⃗B + ⃗A  — you get the same resultant either way.
⃗A first, then ⃗B
⃗B first, then ⃗A
Both paths produce the same resultant ⃗R — this is a parallelogram!
