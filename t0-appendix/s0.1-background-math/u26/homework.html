Part F — Reflection (Sample Answers)

Q20: Identify which two sides are known. If Opposite and Hypotenuse are known, use sin⁻¹. If Adjacent and Hypotenuse are known, use cos⁻¹. If Opposite and Adjacent are known, use tan⁻¹. The mnemonic SOH-CAH-TOA helps remember which ratio each function uses.
Q21: sin(30°) = x/y. The inverse and regular functions undo each other. If sin⁻¹(x/y) = 30°, that means "the angle whose sine is x/y equals 30°", so sin(30°) = x/y. They are inverses of each other.
