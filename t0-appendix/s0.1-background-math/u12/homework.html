Homework — Multi-Step Uncertainty

True/False, multi-step physics problems, and an extended challenge question

Rule 1 — Add/Subtract
y = a ± b
Δy = Δa + Δb
(add ABSOLUTE uncert.)
Rule 2 — Multiply/Divide
y = (a×b)/c
Δy/y = Δa/a + Δb/b + Δc/c
(add FRACTIONAL uncert.)
Rule 3 — Powers & Roots
y = an
Δy/y = |n × Δa/a|
(multiply frac. uncert. by |n|)
▢

Part A — True or False (6 questions)

1
If y = 3x², the percentage uncertainty in y is 6 times the percentage uncertainty in x.
False. The constant 3 is exact and contributes zero uncertainty. Rule 3 with n = 2: %uncert(y) = 2 × %uncert(x). It is twice, not six times, the uncertainty in x.
2
For the formula y = √x, if x has 20% uncertainty, then y has 10% uncertainty.
True. √x = x1/2, so Rule 3 gives %uncert(y) = ½ × 20% = 10%. Square roots halve the percentage uncertainty.
3
In the formula y = a²/b², if a and b are the same measurement (a = b), then the percentage uncertainty in y is zero.
True. If a and b come from the same measurement, y = a²/a² = 1 (a pure constant). Correlated variables cancel in ratios. The uncertainty is 0%.
4
To propagate uncertainty through y = a + b, you add the fractional (relative) uncertainties Δa/a + Δb/b.
False. Rule 1 says you add ABSOLUTE uncertainties: Δy = Δa + Δb. Adding fractional uncertainties applies only to multiplication/division (Rule 2).
5
The formula T = 2π√(L/g) — if only L has uncertainty, then the percentage uncertainty in T equals half the percentage uncertainty in L.
True. 2π is exact. g is treated as exact. So %uncert(T) = ½ × %uncert(L/g) = ½ × %uncert(L). A 4% uncertainty in L produces 2% uncertainty in T.
6
For y = (a²b)/c, the percentage uncertainty in y is: 2×%a + %b + %c.
True. First apply Rule 3 to a²: %uncert(a²) = 2×%a. Then apply Rule 2 to multiply/divide: %uncert(y) = 2×%a + %b + %c.

Electrical Power Dissipation
Harder Rule 3 Rule 2 Two-formula approach
A resistor has resistance R = 100 Ω ± 5% and current through it is I = 0.50 A ± 4%. Power is given by P = I²R.
(a) Find the percentage uncertainty in I².
(b) Find the percentage uncertainty in P.
(c) If P = 25 W, express P with absolute uncertainty.
(d) Alternatively, using V = IR: if V = 50 V ± ?, what is the percentage uncertainty in V? Use %uncert(V) = %uncert(I) + %uncert(R) to find it. Then verify P = V²/R gives the same result for %uncert(P).
R: 5%
I: 4%
P = I²R
Your Work Space
(a) %uncert(I²) =
(b) %uncert(P) =
(c) P = 25 ± ? W
(d) %uncert(V) =

✅ Full Solution

a
I²: Rule 3, n = 2. %uncert(I²) = 2 × 4% = 8%
b
P = I² × R. Rule 2:
%uncert(P) = %uncert(I²) + %uncert(R) = 8% + 5% = 13%
c
Absolute: ΔP = 13% × 25 W = 3.25 W ≈ 3 W
P = 25 ± 3 W
d
V = IR. Rule 2: %uncert(V) = 4% + 5% = 9%
Now P = V²/R. Rule 3 then Rule 2:
%uncert(V²) = 2 × 9% = 18%
%uncert(P) = %uncert(V²) + %uncert(R) = 18% + 5% = 23%?
Wait — V and R are NOT independent (V = IR, so V depends on R). This is a correlated variable trap! The two formulas are equivalent physical expressions, but P = I²R is the correct one to use when I and R are your independent measurements. The V²/R route double-counts R. Answer confirmed: %uncert(P) = 13%.
%uncert(P) = 13%  |  P = 25 ± 3 W

Use the rule for small changes: Δ(x²) ≈ 2xΔx (derived from Rule 3 in absolute form).

Part C — Extended Challenge

Escape Velocity — A Multi-Layer Chain

The escape velocity from a planet is given by vesc = √(2GM/R), where G = 6.674 × 10³&sup8; m³kg¹s² is exact (defined constant), M is the planet mass, and R is the planet radius.

A student measures the following for an unknown planet:

M = 4.0 × 10²&sup4; kg ± 8%
R = 6.5 × 10&sup6; m ± 3%
G = exact constant
(a) What is the percentage uncertainty in the product GM?
(b) What is the percentage uncertainty in 2GM/R? (Note: 2 is exact)
(c) What is the percentage uncertainty in vesc = √(2GM/R)?
(d) The calculated vesc = 9,050 m/s. Express vesc with its absolute uncertainty.
(e) Conceptual: if you doubled the uncertainty in M to 16%, how would %uncert(vesc) change? Explain without recalculating.

✅ Full Solution

(a) GM: G is exact (0% uncertainty). Rule 2 — multiply: %uncert(GM) = 0% + 8% = 8%
(b) 2GM/R: 2 is exact. Rule 2 — divide: %uncert(2GM/R) = %uncert(GM) + %uncert(R) = 8% + 3% = 11%
(c) vesc = √(2GM/R). Square root = power n = ½. Rule 3:
%uncert(v) = ½ × 11% = 5.5%
(d) Absolute uncertainty: Δv = 5.5% × 9050 = 498 m/s ≈ 500 m/s
vesc = 9050 ± 500 m/s
(e) If %uncert(M) doubles to 16%: %uncert(2GM/R) = 16% + 3% = 19% (not double, because R still contributes 3%). Then %uncert(v) = ½ × 19% = 9.5%. So vesc's uncertainty increases from 5.5% to 9.5% — roughly a 73% increase, not a doubling. R's independent contribution “dampens” the effect of doubling M's uncertainty.
v_esc = 9050 ± 500 m/s  (5.5% uncertainty)
