Propagating Uncertainty

Combining all three rules to solve real physics problems — area, density, ratios, and oscillations

📋

The Three Rules — Quick Reference

All Three Uncertainty Propagation Rules
Rule 1
Addition & Subtraction
If  y = a ± b
Δy = Δa + Δb
Add ABSOLUTE uncertainties
Rule 2
Multiplication & Division
If  y = (a×b)/c
Δy/y = Δa/a + Δb/b + Δc/c
Add FRACTIONAL uncertainties
Rule 3
Powers & Roots
If  y = an
Δy/y = |n × Δa/a|
Multiply fractional uncert. by |n|
Key Strategy: Break multi-step physics formulas into stages. Apply the matching rule at each stage, then chain the results together.

Real physics problems rarely involve just one operation. A formula like density = mass / volume requires you to first calculate volume (which may involve cubing a length), then divide by mass. Each step introduces uncertainty.

The skill in this unit is chaining rules: identify the mathematical structure at each stage, pick the right rule, and propagate the uncertainty step by step until you reach the final answer.

An important concept: exact constants have zero uncertainty. The π in πr² is exact, so it contributes nothing to the fractional uncertainty. Only measured quantities carry uncertainty.

Also watch for correlated variables — if the same measurement appears in both numerator and denominator, the uncertainties partially cancel rather than add.

4 Fully Worked Examples

Example 1
Area of a Circular Plate
Example 1
Powers Rule · constant π contributes nothing
A circular metal plate has radius r with a fractional uncertainty of Δr/r. Find the percentage uncertainty in the area A = πr².
A = πr²
uncertainty: Δr/r
π = exact constant
1
Identify all factors: π is an exact constant (zero uncertainty); r² is a measured quantity raised to the power 2.
2
Since multiplication by an exact constant does not change fractional uncertainty, focus on r². Rule 3
3
Apply the power rule with n = 2:
Δ(r²) / r² = |2 × Δr/r| = 2 × Δr/r
4
Since A = π × r² and π is exact: ΔA / A = Δ(r²) / r² = 2 × Δr/r
Final Answer
ΔA / A = 2 × Δr / r
The percentage uncertainty in area is exactly twice the percentage uncertainty in radius.
Why? π is a mathematical constant — it has infinitely many decimal places and contributes zero uncertainty. The only source of uncertainty is the measured radius r. Because we square it, the power rule multiplies the fractional uncertainty by 2. If your radius had 5% uncertainty, the area would have 10% uncertainty.
Example 2
Density of a Cube
Example 2
Powers Rule + Division Rule · two-stage chain
A cube has side length L with a 4% uncertainty. Its mass is 5 kg ± 8%. Find the percentage uncertainty in its density ρ = m / L³.
L: 4% uncertainty
m: 8% uncertainty
ρ = m / L³
1
Stage A — Find uncertainty in Volume: Volume = L³ → apply the power rule with n = 3. Rule 3
2
%uncert(V) = |3| × %uncert(L) = 3 × 4% = 12%
3
Stage B — Find uncertainty in Density: ρ = m / V → apply the division rule: add fractional uncertainties. Rule 2
4
%uncert(ρ) = %uncert(m) + %uncert(V) = 8% + 12% = 20%
Final Answer
Density uncertainty = 20%
Rule 3 → Rule 2 chain: cube the length uncertainty, then add to mass uncertainty.
Chain logic: First apply Rule 3 to get the volume uncertainty (3 × 4% = 12%). Then treat density as a division problem — mass divided by volume — and apply Rule 2 (add fractional uncertainties): 8% + 12% = 20%.
Example 3
Ratio of Volumes — Sphere Inside a Cube
Example 3
Correlated variables · ratio of two expressions sharing the same measurement
A sphere fits exactly inside a cube. The diameter of the sphere equals the side of the cube: d = 20.0 ± 0.8 m. The sphere has radius r = d/2. Find the percentage uncertainty in the ratio Vsphere / Vcube = (4/3)πr³ / d³.
d = 20.0 ± 0.8 m
r = d/2
ratio = V₃ / V□
1
Simplify the ratio algebraically first: V₃ / V□ = (4/3)πr³ / d³ Substitute r = d/2: = (4/3)π(d/2)³ / d³ = (4/3)π × d³/8 / d³
2
The d³ terms cancel completely: ratio = (4/3)π/8 = π/6 ≈ 0.5236
3
Since the ratio simplifies to the pure constant π/6 (containing no measured quantities), its uncertainty is zero.
4
Physical interpretation: Both volumes depend on the same single measurement d. Because they are perfectly correlated, any change in d scales both volumes by the same factor — the ratio stays constant regardless of the exact value of d.
Final Answer
% uncertainty in ratio = 0%
The ratio V₃/V□ = π/6 is a pure constant — it carries zero uncertainty.
Key lesson — correlated variables: When the same measured quantity appears in both numerator and denominator, simplify the algebra before propagating uncertainties. The “naïve” approach of treating numerator and denominator as independent would give 12% + 12% = 24% — a wrong answer. Since both volumes depend on the same d, they move together and the ratio is locked at π/6.
Example 4
Angular Frequency — Division then Square Root
Example 4
Division Rule + Root Rule (Rule 2 then Rule 3) · three-step chain
A spring oscillates with maximum acceleration amax ± 20% and initial displacement x0 ± 12%. The angular frequency is given by k = √(amax / x0). Find the percentage uncertainty in k.
amax: 20% uncertainty
x0: 12% uncertainty
k = √(amax/x0)
1
Stage A — Uncertainty in the ratio: amax/x0 is a division → add fractional uncertainties. Rule 2
2
%uncert(amax/x0) = 20% + 12% = 32%
3
Stage B — Uncertainty after square root: Taking √ means raising to the power n = ½. Apply the power rule. Rule 3
4
%uncert(k) = |½| × 32% = 16%
Final Answer
%uncertainty in k = 16%
Rule 2 then Rule 3: divide → then take square root → halve the fractional uncertainty.
Chain logic: First find the fractional uncertainty of the expression inside the square root using Rule 2 (20% + 12% = 32%). Then apply Rule 3 with n = ½ (square root = power of ½), which halves the fractional uncertainty: ½ × 32% = 16%.
⚠

Special Cases & Common Mistakes

⚠ Correlated Variables — Always Simplify Algebra First

If the same measured quantity appears in multiple places in a formula, simplify algebraically before propagating. Variables that share the same measurement are correlated — they cannot be treated as independent sources of uncertainty.

Example: Vsphere/Vcube — both depend on d. Simplify first to get π/6, then realize the uncertainty is 0.

Rule: simplify → then propagate. Never propagate first and simplify later.
✓

Exact Constants

π, e, 4/3, 2, ½ — any pure number in a formula has zero uncertainty.

✗

Do NOT Add Uncorrelated Uncertainties

Adding uncertainties from the same variable twice is a common error. Check each variable appears only once after simplification.

√

Square Root = Power of ½

√x = x1/2 so Δ(√x)/√x = ½ × Δx/x. Roots reduce uncertainty.
