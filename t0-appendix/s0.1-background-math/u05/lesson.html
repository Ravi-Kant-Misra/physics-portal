Converting Units

Master the Factor-Label Method — a foolproof system for converting any unit to any other

💡

What Does It Mean to Convert a Unit?

When you convert a unit, you are rewriting the same measurement using a different unit. The actual physical quantity doesn't change — only the way it's written does.

For example: 6 feet and 72 inches describe the same height. They represent the same value, just written with different units. This is possible because 1 foot = 12 inches, so you can multiply by the fraction 12 inches/1 foot without changing the actual measurement.

The Core Idea

6 feet × (12 inches / 1 foot) = 72 inches

The two measurements represent the same value — they're just written with different units. Multiplying by a conversion factor always equals multiplying by 1.

Why This Works — The Conversion Factor

Each fraction you write is called a conversion factor. Because the numerator equals the denominator (just written in different units), the fraction equals exactly 1. Multiplying by 1 never changes the actual value — only the unit.

( 12 cars / 1 bus ) → top = bottom in real units → = 1

a × 1 × 1 × 1 = a  — you can chain as many conversion factors as needed!
