Recording the Angle of a Vector

In-class practice — compass notation, equivalent forms, and drawing vectors

Instructions

Part ALabel the angle shown in each compass diagram using correct notation.
Part BConvert between equivalent forms of the same vector direction.
Part CDraw vectors on blank compasses and reveal the answer key.
🏷️

Part A — Label the Angle

For each compass diagram, write the correct angle notation in the answer box. The angle arc shows which axis is the reference.

Part B — Convert Between Equivalent Forms

Each vector has two valid notations. Given one form, find the other. Remember: the two angles always add up to 90°.

Convert: 30° North of East
Write the equivalent "_ of North/South" form.
B2
Convert: 72° West of North
Write the equivalent "_ of East/West" form.
B3
Convert: 48° South of West
Write the equivalent "_ of South/North" form.
B4
Which pair of notations describes the same vector?
Select the matching pair.
✏️

Part C — Draw the Vector

Draw a vector on the blank compass that matches the given notation. Then reveal the answer to check your work.
