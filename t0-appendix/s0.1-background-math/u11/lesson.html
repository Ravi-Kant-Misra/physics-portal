Propagating Uncertainty

Powers & Roots — when you raise a measurement to a power, the uncertainty scales with it

📐

The Rule

The Power Rule for Uncertainty
If  y = an,  then  Δy / y = | n × Δa / a |
When you raise a measurement with uncertainty to a power, the fractional uncertainty of the result equals the absolute value of the power multiplied by the original fractional uncertainty.
Fractional uncertainty of result = |n| × (fractional uncertainty of base)
Alternative form:  (a ± Δa)n = an ± | n × Δa/a | × an

In previous units you learned how to propagate uncertainty through addition/subtraction (add absolute uncertainties) and multiplication/division (add fractional uncertainties). Powers are just a special case of multiplication.

Think of squaring a number as multiplying it by itself twice: a² = a × a. Each multiplication adds the fractional uncertainty once — so squaring doubles it. Cubing adds it three times, so the fractional uncertainty triples.

For roots, remember that a square root is just raising to the power ½, a cube root is the power ⅓, and so on. The same rule applies — the fractional uncertainty is multiplied by the (now fractional) power.

🔑
Why the Absolute Value?
The absolute value |n| ensures the uncertainty is always positive, regardless of whether the power is positive or negative.

For example, a⁻² and a² produce the same fractional uncertainty — you lose the same relative precision either way.
💡

4 Key Insights

✕2

Squaring DOUBLES the fractional uncertainty

If your measurement has 10% fractional uncertainty, then squaring it gives a result with 20% fractional uncertainty.

|n| = 2 → Δy/y = 2 × Δa/a
✕3

Cubing TRIPLES the fractional uncertainty

Raising to the 3rd power multiplies the fractional uncertainty by 3. Precision degrades rapidly with large powers.

|n| = 3 → Δy/y = 3 × Δa/a
÷2

Square rooting HALVES the fractional uncertainty

Taking a square root (n = ½) cuts the fractional uncertainty in half — roots improve relative precision.

|n| = ½ → Δy/y = ½ × Δa/a
|n|

The absolute value — sign of n doesn't matter

Whether n is +2 or −2, the fractional uncertainty scales by 2. Negative powers (like a⁻¹) still increase uncertainty.

|+n| = |−n| → same fractional uncertainty

√ Roots Are Just Fractional Powers

Any root can be written as a fractional exponent. This means the same rule applies automatically:

≢√x = x1/2 → multiply Δx/x by 1/2
⁴√x = x1/4 → multiply Δx/x by 1/4
ⁿ√x = x1/n → multiply Δx/x by 1/n

Worked Examples

Powers
Raising to integer powers
Example 1
n = 2 · squaring · absolute uncertainty given
(5 ± 2 kg)2
1
Compute the base value: 5² = 25 kg²
2
Find the fractional uncertainty of the base: Δa/a = 2/5 = 0.4
3
Apply the rule — multiply by |n| = 2: Δy/y = 2 × 0.4 = 0.8
4
The fractional uncertainty of the result is 0.8 (or 80%)
Final Answer
25 kg² ± 0.8 (fractional)
Note: absolute uncertainty = 0.8 × 25 = 20 kg²
Example 2
n = 4 · fourth power
(2.5 ± 0.5 m)4
1
Base value: 2.5⁴ = 39.1 m⁴
2
Fractional uncertainty: Δa/a = 0.5/2.5 = 0.20
3
Apply rule with |n| = 4: Δy/y = 4 × 0.20 = 0.80
Final Answer
39.1 m⁴ ± 0.80 (fractional)
Notice how a 20% fractional uncertainty in the base becomes an 80% fractional uncertainty in the 4th power — large powers amplify uncertainty dramatically!
Example 3
n = 3 · percentage uncertainty given
(20 N ± 10%)3
1
Base value: 20³ = 8000 N³
2
Fractional uncertainty is already given as 10%
3
Apply rule with |n| = 3: Δy/y = 3 × 10% = 30%
Final Answer
8000 N³ ± 30%
When the uncertainty is given as a percentage, the rule is especially clean: just multiply the percentage by |n|. 10% cubed becomes 30%.

Square roots, cube roots, nth roots
Example 5
n = ½ · square root
√(10 ± 2 V)
i.e. (10 V)1/2
1

Written as: (10 V)½ ± (½ × 0.2) V½ = 3.2 V½ ± 0.1
The square root halved the fractional uncertainty from 20% down to 10%. Taking roots is the one case where the uncertainty actually improves relative to the base.

Quick Reference Summary

Operation Power n Effect on fractional uncertainty Example
a² n = 2 Doubles: Δy/y = 2 × Δa/a 5% → 10%
a³ n = 3 Triples: Δy/y = 3 × Δa/a 5% → 15%
a⁴ n = 4 Quadruples: Δy/y = 4 × Δa/a 5% → 20%
√a (a½) n = ½ Halves: Δy/y = ½ × Δa/a 10% → 5%
3√a (a⅓) n = ⅓ Reduces to ⅓: Δy/y = ⅓ × Δa/a 9% → 3%
4√a (a¼) n = ¼ Reduces to ¼: Δy/y = ¼ × Δa/a 8% → 2%
a−1 n = −1 |n| = 1: no change in magnitude 5% → 5%
a−2 n = −2 |n| = 2: doubles (same as squaring) 5% → 10%

Step-by-step procedure for any power/root problem:

  1. Calculate the base value: compute an
  2. Find the fractional uncertainty of the base: Δa / a
  3. Multiply by |n|: Δy/y = |n| × Δa/a
  4. State the result as: an ± (fractional uncertainty) — or convert to absolute if required
