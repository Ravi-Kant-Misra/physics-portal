Rules Reminder

If y = an, then Δy/y = |n × Δa/a|
  1. Compute the base value: evaluate an.
  2. Find the fractional uncertainty of the base: Δa / a.
  3. Multiply by |n| to get the fractional uncertainty of the result.
  4. For roots: a square root is n = ½; a cube root is n = ⅓; an nth root is n = 1/n.
Key insight: Squaring doubles the fractional uncertainty. Square rooting halves it. The sign of n never matters — only |n| counts.

Part D — Extended Problems

Q13. V = (4/3)π(5.0)³ = (4/3)π(125) = 524 cm³.
Since V ∝ r³, we have ΔV/V = 3 × Δr/r = 3 × (0.5/5.0) = 3 × 0.10 = 0.30 (30%).
Absolute uncertainty = 0.30 × 524 ≈ 157 cm³. Final: V = 524 ± 157 cm³ (or 524 cm³ ± 30%).
Q14. T ∝ √L, so n = ½ for L. ΔL/L = 0.010/0.250 = 0.040 (4.0%).
ΔT/T = ½ × 4.0% = 2.0%. The percentage uncertainty in T is 2.0%.
Q15. She wants to reduce 24% to 6%. That is a factor of 24/6 = 4 reduction.
Δy/y = |n| × Δa/a → |n| = (Δy/y) / (Δa/a) = 6%/24% = 1/4.
So n = 1/4, which is the fourth root (⁴√). It is a fractional power, not an integer.
Q16. Student A: Δa/a = 0.4/8.0 = 0.05 = 5%. Student B: Δa/a = 4% directly.
Wait — 0.4/8.0 = 0.050 = 5%, and Student B gives 4% — these are not the same. The difference is that 0.4/8.0 = 5%, not 4%. So they do not give the same result.
Student A Δy/y = 2 × 5% = 10%. Student B Δy/y = 2 × 4% = 8%. The two reports differ because the fractional uncertainties of the base measurements are different (5% vs 4%).
