Reference — The Rule

For addition (y = a + b):
Δy = Δa + Δb
For subtraction (y = a − b):
Δy = Δa + Δb  (same rule!)
Percent → Absolute first:
Δa = (% / 100) × a
Fractional → Absolute first:
Δa = (frac) × a
Uncertainties ALWAYS add Round Δ to 1 sig fig Match decimal places in final answer ⚠️ Never subtract uncertainties!

Values: 4.5+3.1=7.6. Uncertainties: 0.2+0.4=0.6.

Part D — Short Answer

Question 13: Total length = 45 + 45 = 90 cm. Total uncertainty = 3 + 3 = 6 cm. Answer: 90 ± 6 cm. The percentage uncertainty of each wire was 3/45 = 6.7%, but the combined result is also 6/90 = 6.7% — the relative uncertainty has stayed the same. Adding identical measurements does not improve your relative uncertainty.
Question 14: The student made an error by subtracting the uncertainties (2 − 1 = 1). This is wrong because uncertainty represents a worst-case range — both uncertainties can act in the same direction at once, so they must always be added. The correct answer is: values 8 − 3 = 5 kg, uncertainties 2 + 1 = 3 kg, so the answer is 5 ± 3 kg.
Question 15: Difference in values: 340 − 336 = 4 m/s. Uncertainty: 2 + 8 = 10 m/s. Answer: 4 ± 10 m/s. The uncertainty (10 m/s) is much larger than the difference (4 m/s). This means the difference could be anywhere from −6 to +14 m/s, which includes zero. We cannot conclude the experiments gave different results — the results are consistent with each other within experimental uncertainty.
