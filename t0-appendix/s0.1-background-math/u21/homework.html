Answer Key

Part A — Real-World Scenarios

Q1 (Plane NE):vx: right (+), vy: up (+). Both non-zero (diagonal vector, Quadrant I)
Q2 (Sub down):vx = 0 (no horizontal travel). vy: down (−), vy = 5 m/s downward
Q3 (Ball SW):dx: left (−), dy: down (−). dx ≠ 0. Quadrant III
Q4 (Train left):vx = −80 km/h. vy = 0. Flat track = no vertical motion

Part B — Pythagorean Calculations

Q5:9² + 12² = A² → 81 + 144 = 225 → A = 15 m (9-12-15 = 3×3-4-5)
Q6:vx² = 26² − 10² = 676 − 100 = 576 → vx = 24 m/s (10-24-26 = 2×5-12-13)
Q7:20² + 21² = F² → 400 + 441 = 841 → F = 29 N (20-21-29 Pythagorean triple)
Q8:dy² = 50² − 30² = 2500 − 900 = 1600 → dy = 40 m (30-40-50 = 10×3-4-5)

Part C — Concept Questions

Q9:Ax and Ay are perpendicular, so they form a right triangle with A as hypotenuse. The correct relationship is Ax² + Ay² = A², not A = Ax + Ay.
Q10:Ay measures vertical travel. A horizontal vector moves purely sideways — it has zero vertical displacement, so Ay = 0.
Q11:The arrow means direction matters. →Ax and →Ay are added tip-to-tail (vectorially), not as plain numbers. Their directions are perpendicular, which is why we need the Pythagorean theorem for magnitudes.

Part D — True / False

Q12:TRUE — Ax = 0 means no horizontal component, so the vector must be purely vertical.
Q13:TRUE — x and y are always perpendicular by definition, so the Pythagorean theorem always applies.
Q14:FALSE — 3² + 4² = 5² → A = 5 m, not 7 m. Never add scalar components directly.
Q15:TRUE — Diagonal means traveling in both horizontal and vertical directions simultaneously.
Q16:FALSE — Bottom-left (Quadrant III) has Ax pointing LEFT (negative) and Ay pointing DOWN (also negative).
Q17:TRUE — A is the hypotenuse and is always the longest side of a right triangle (A ≥ Ax and A ≥ Ay).
