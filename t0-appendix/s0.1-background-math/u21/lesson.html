X and Y Components
of Vectors

Decomposing any vector into its horizontal and vertical parts

📋

What Are Vector Components?

💡

Big Idea

Every vector — no matter which direction it points — can be broken into exactly two perpendicular parts: one pointing purely horizontal (x) and one pointing purely vertical (y). These parts are called the components of the vector.

→ X-Component (Aˇₓ)

How far the vector travels in the horizontal direction — left or right. X-components are always drawn as horizontal arrows.
Left ← or Right →

↑ Y-Component (Aˇᵧ)

How far the vector travels in the vertical direction — up or down. Y-components are always drawn as vertical arrows.
Down ↓ or Up ↑

Official Notation

→AThe original (resultant) vector
→AˇₓX-component of →A (horizontal part)
→AˇᵧY-component of →A (vertical part)
Aˇₓ & AˇᵧScalar magnitudes (just the size)

Bird Flying Diagonally — Decomposed into Components

The diagonal vector →A is made up of a red horizontal part (→Ax) and a blue vertical part (→Ay).

🎯

Special Cases: Horizontal, Vertical, Diagonal

Vector Direction Ax value Ay value Reason
→ Purely Horizontal Ax = A Ay = 0 No vertical travel at all
↑ Purely Vertical Ax = 0 Ay = A No horizontal travel at all
↗ Diagonal (any angle) Ax ≠ 0 Ay ≠ 0 Travels both horizontally and vertically
📌

Key Insight

A horizontal vector has all of its magnitude in Ax and none in Ay. A vertical vector is the opposite. Only a diagonal vector shares its magnitude between both components.

✈

Vectors in All Four Quadrants

Every quadrant produces a different combination of x and y component directions. The components are always perpendicular to each other.

Quadrant I — Top-Right

Ax points RIGHT (+) Ay points UP (+)

Quadrant II — Top-Left

Ax points LEFT (−) Ay points UP (+)

Quadrant III — Bottom-Left

Ax points LEFT (−) Ay points DOWN (−)

Quadrant IV — Bottom-Right

Ax points RIGHT (+) Ay points DOWN (−)
📋

The 2 Fundamental Rules

1

Vector Addition Rule

→A = →Ax + →Ay

Every vector equals the vector sum of its x-component and y-component. This is a vector equation — the arrows matter. You add them tip-to-tail and they reconstruct the original vector.

Think of it as: "The original vector IS its two perpendicular parts combined."

2

Pythagorean Theorem Rule

Ax² + Ay² = A²

The scalar magnitudes of the components satisfy the Pythagorean theorem, because Ax and Ay are always perpendicular to each other. This lets you find any missing magnitude if you know the other two.

Note: A is the hypotenuse; Ax and Ay are the two legs of a right triangle.

⚠️

Common Mistake — Do NOT write this!

A = Ax + Ay  ← This is wrong for most vectors! Scalar magnitudes do NOT add like regular numbers unless the vectors happen to be along the same axis. Always use the Pythagorean theorem: Ax² + Ay² = A²

🔨

Worked Examples

1

Horizontal Vector — A = 5 m horizontally to the right

Given: Vector →A points purely to the right. Magnitude A = 5 m.
  • The vector is horizontal, so it has no vertical travel. Therefore Ay = 0.
  • All of the magnitude is in the horizontal direction: Ax = 5 m.
  • Check with Pythagorean theorem: 5² + 0² = 5² → 25 = 25 ✓
Ax = 5 m  (right)   |   Ay = 0 m
2

Pythagorean Check — A = 5 m, Ax = 3 m, Ay = 4 m

Given: A diagonal vector with Ax = 3 m and Ay = 4 m. Find the magnitude A and verify.
  • Apply Rule 2: Ax² + Ay² = A²
  • Substitute: 3² + 4² = A² → 9 + 16 = A²
  • Solve: A² = 25 → A = 5 m
  • Check: 3² + 4² = 9 + 16 = 25 = 5² ✓ (the classic 3-4-5 right triangle)
A = 5 m   |   3² + 4² = 5²  ✓
3

Find Missing Component — A = 13 m, Ax = 5 m, find Ay

Given: Vector magnitude A = 13 m, x-component Ax = 5 m. Find Ay.
  • Write Rule 2: Ax² + Ay² = A²
  • Rearrange for Ay: Ay² = A² − Ax²
  • Substitute: Ay² = 13² − 5² = 169 − 25 = 144
  • Take square root: Ay = 12 m (the 5-12-13 Pythagorean triple)
Ay = 12 m   |   5² + 12² = 13²  ✓

Question 1
A vector points straight up (vertically). What are its components?
AAx = A, Ay = 0
BAx = 0, Ay = A
CAx = A/2, Ay = A/2
DAx = Ay = 0
✓ Correct! A vertical vector has Ax = 0 (no horizontal travel) and Ay = A (all vertical).
✗ Not quite. A vertical vector moves only up/down — so there is no horizontal component (Ax = 0) and all of the magnitude is vertical (Ay = A).
Question 2
If Ax = 6 m and Ay = 8 m, what is the magnitude A?
AA = 14 m
BA = 7 m
CA = 10 m
DA = 48 m
✓ Correct! 6² + 8² = 36 + 64 = 100 = 10², so A = 10 m. Classic 6-8-10 (scaled 3-4-5) triple!
✗ Use the Pythagorean theorem: Ax² + Ay² = A² → 36 + 64 = 100 → A = 10 m. Remember: you CANNOT just add Ax + Ay!
Question 3
Vector →A points diagonally to the bottom-right. Which statement about its components is correct?
AAx points right, Ay points up
BAx points right, Ay points down
CAx points left, Ay points down
DAx = 0, Ay points down
✓ Correct! Bottom-right = Quadrant IV. Horizontal part goes right (+Ax), vertical part goes down (−Ay).
✗ Bottom-right is Quadrant IV. The x-component always matches horizontal direction (right = +), and y-component matches vertical (down = −).
