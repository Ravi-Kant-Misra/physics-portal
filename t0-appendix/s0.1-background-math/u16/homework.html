Answer Key

Part A — Q1: Spring Experiment

Calculations:
Average slope = (21.2 + 15.6) / 2 = 36.8 / 2 = 18.4 N/m
Slope uncertainty = (21.2 − 15.6) / 2 = 5.6 / 2 = 2.8 N/m
% slope unc = (2.8 / 18.4) × 100% ≅ 15.2%
Average y-int = (−1.8 + 2.4) / 2 = 0.6 / 2 = 0.3 N
Δb = |2.4 − (−1.8)| / 2 = |4.2| / 2 = 2.1 N
% y-int unc = (2.1 / 0.3) × 100% = 700% (very large — b is near zero)
Average best-fit equation:
Force(N) = ( 18.4 ± 2.8 N/m ) × stretch(m) + ( 0.3 ± 2.1 N )
Max slope equation:
Force(N) = ( 21.2 N/m ) × stretch(m) + ( −1.8 N )
Min slope equation:
Force(N) = ( 15.6 N/m ) × stretch(m) + ( 2.4 N )
Spring constant: k = 18.4 ± 2.8 N/m (same as slope, since F = kx)

Part A — Q2: Pendulum T² vs L

Calculations:
Average slope = (4.22 + 3.70) / 2 = 7.92 / 2 = 3.96 s²/m
Slope uncertainty = (4.22 − 3.70) / 2 = 0.52 / 2 = 0.26 s²/m
% slope unc = (0.26 / 3.96) × 100% ≅ 6.6%
Average y-int = (−0.14 + 0.30) / 2 = 0.16 / 2 = 0.08 s²
Δb = |0.30 − (−0.14)| / 2 = 0.44 / 2 = 0.22 s²
Average best-fit equation:
Period²(s²) = ( 3.96 ± 0.26 s²/m ) × Length(m) + ( 0.08 ± 0.22 s² )
Finding g:
slope = 4π² / g → g = 4π² / 3.96 = 39.48 / 3.96 ≅ 9.97 m/s²
% unc in g = % unc in slope = 6.6%
Δg = 0.066 × 9.97 ≅ 0.66 m/s²
g = 9.97 ± 0.66 m/s² (or rounded to 10.0 ± 0.7 m/s²)

Part B — Q3: Slope and Y-intercept Units

Force (N) vs acceleration (m/s²): slope unit = N/(m/s²) = kg  |  y-int unit = N
Voltage (V) vs Current (A): slope unit = V/A = Ω (Ohms)  |  y-int unit = V
KE (J) vs v² (m²/s²): slope unit = J/(m²/s²) = kg  |  y-int unit = J
Period² (s²) vs Length (m): slope unit = s²/m  |  y-int unit = s²

Part C — True / False

4. TRUE — Average slope = (m_max + m_min) / 2 ✓
5. FALSE — The maximum slope line has the lowest y-intercept (it pivots steeply, forcing its intercept down). The minimum slope line has the highest y-intercept.
6. TRUE — Slope unit = y-unit / x-unit = m/s ✓ (y in meters, x in seconds)
7. FALSE — When writing the max slope equation, use bmax (the y-intercept from the max slope line itself), not the average y-intercept. Each individual line equation uses that line's own slope and intercept.
8. TRUE — A negative y-intercept is physically acceptable (e.g., the line crosses the y-axis below zero). The uncertainty value (positive) is also correct. ✓

Q10: When the slope increases (steeper line), the line must still pass through the same cluster of data points. To be steeper while passing through the middle of the data, the line pivots: its left side drops and its right side rises. This means the y-intercept (where the line hits the y-axis, which is to the left of the data) goes down as the slope goes up. Therefore mmax corresponds to bmax being the lowest intercept value, and mmin corresponds to bmin being the highest. Since bmin > bmax, (bmin − bmax) is positive. We write it this way (rather than bmax − bmin) to get a positive result without needing the absolute value. However, this relationship can reverse in some data configurations, which is why the formula uses |bmin − bmax| to guarantee a positive uncertainty regardless.
