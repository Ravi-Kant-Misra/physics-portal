Line of Best Fit Equations

Reading Logger Pro output, calculating slope and y-intercept uncertainties, and writing the complete IB Physics best-fit equation

💻

What Logger Pro Gives You

After you add three best-fit lines in Logger Pro (best, maximum slope, minimum slope), each line gives you two numbers: m (slope / gradient) and b (y-intercept). These appear in the equation box on the graph as y = mt + b.

Your job is to take these six numbers (slope and intercept for each of the three lines) and use them to calculate the average slope, slope uncertainty, average y-intercept, and y-intercept uncertainty — then write the full formatted equation that IB Physics requires.

The three lines give you six numbers

Best-fit line: mbest, bbest  |  Max slope line: mmax, bmax  |  Min slope line: mmin, bmin. The averages of the max and min values give your reported slope, intercept, and their uncertainties.

📐

The Average Best-Fit Equation Format

Colour-coded template — every part has a role

y-variable(y-unit) = ( slope ± slope-unc slope-unit/x-unit ) × x-variable(x-unit) + ( y-int ± y-int-unc y-unit )
y-variable

Name of the dependent variable (e.g. displacement, velocity)

y-unit

Unit of the y-variable in parentheses (e.g. m, m/s, N)

slope

Average slope = (mmax + mmin) / 2

slope-unc

Absolute uncertainty = (mmax − mmin) / 2

slope-unit/x-unit

Ratio of y-unit to x-unit (e.g. m/s, N/m, kg·m/s²)

x-variable

Name of the independent variable (e.g. time, length)

x-unit

Unit of the x-variable (e.g. s, m, kg)

y-int

Average y-intercept = (bmax + bmin) / 2

y-int-unc

|Absolute uncertainty| = |bmin − bmax| / 2

Watch the sign for y-intercept uncertainty

Slope: uncertainty = (mmax − mmin) / 2 — always positive because mmax > mmin.
Y-intercept: the max slope line tends to have a lower y-intercept. So you may see bmin > bmax, making (bmin − bmax) positive. Always take the absolute value to get a positive uncertainty: Δb = |bmin − bmax| / 2.

🧮

Calculating Slope and Y-Intercept Uncertainties

Slope Calculations

Average slope = (m_max + m_min) / 2 Δslope = (m_max − m_min) / 2 %unc = (Δslope / avg) × 100%

mmax is always larger, so the result is always positive.

Y-Intercept Calculations

Average b = (b_max + b_min) / 2 Δb = |b_min − b_max| / 2 %unc = (Δb / |avg b|) × 100%

Take absolute value — bmin can be larger than bmax.

Why max slope = lower y-intercept?

The max slope line is the steepest line that still passes through all error bars. To achieve this, it pivots around the data and tilts upward steeply — which forces its y-intercept down. The min slope line does the opposite: shallower tilt, higher y-intercept. This is why bmax (from the max slope line) is usually the smallest y-intercept value.

📊

Worked Example — Displacement vs Time

Step 1 — Read the Logger Pro output

Best-fit line: slope = 3.2   y-intercept = −0.7
Max slope line: slope = 3.9   y-intercept = −4.9
Min slope line: slope = 2.5   y-intercept = 2.6
Graph axes: x = time (s), y = displacement (m)
Step 2 — Calculate average slope and slope uncertainty
Average slope = (m_max + m_min) / 2 = (3.9 + 2.5) / 2 = 6.4 / 2 = 3.2 m/s Δslope = (m_max − m_min) / 2 = (3.9 − 2.5) / 2 = 1.4 / 2 = 0.7 m/s

So the slope is 3.2 ± 0.7 m/s. Notice the average matches the best-fit slope — this is expected when the max and min lines are drawn symmetrically.

Step 3 — Calculate average y-intercept and y-intercept uncertainty
Average b = (b_max + b_min) / 2 = (−4.9 + 2.6) / 2 = −2.3 / 2 = −1.15 ≅ −0.7 m Δb = |b_min − b_max| / 2 = |2.6 − (−4.9)| / 2 = |7.5| / 2 = 3.75 ≅ 4.3 m

Note: the slide example reports −0.7 ± 4.3 m — the ± uncertainty is large here because the two lines' intercepts are very spread out (from −4.9 to +2.6 m).

Step 4 — Write the full average best-fit equation

Average best-fit equation:

displacement(m) = ( 3.2 ± 0.7 m/s ) × time(s) + ( −0.7 ± 4.3 m )
Step 5 — Write the Max and Min slope equations (using their own m and b)

Maximum slope equation:

displacement(m) = ( 3.9 m/s ) × time(s) + ( −4.9 m )

Minimum slope equation:

displacement(m) = ( 2.5 m/s ) × time(s) + ( 2.6 m )

The max and min equations use the slope and y-intercept from their own specific line — not the averages.

Things to check

✔ y-variable name and unit in first parentheses

✔ Slope and slope-uncertainty with correct combined unit (y/x)

✔ x-variable name and unit in multiplication parentheses

✔ Y-intercept and its uncertainty with y-unit only

✔ Sign of y-intercept preserved (can be negative)

Common mistakes

✘ Using best-fit slope instead of average of max/min

✘ Wrong unit on slope (forgetting to divide y-unit by x-unit)

✘ Using y-unit for slope instead of y-unit/x-unit

✘ Forgetting the absolute value for y-intercept uncertainty

✘ Writing the max/min equations with the average slope
