Quick Reference — Line of Best Fit Equations

y-variable(y-unit) = ( slope ± slope-unc  slope-unit/x-unit ) × x-variable(x-unit) + ( y-int ± y-int-unc  y-unit )
  • Average slope = (m_max + m_min) / 2
  • Slope uncertainty = (m_max − m_min) / 2
  • Average y-intercept = (b_max + b_min) / 2
  • Y-intercept uncertainty = |b_min − b_max| / 2 (absolute value!)
  • % Uncertainty = (absolute uncertainty / average) × 100%

Answer Key

Part A

Q1 — Reading values from Logger Pro
mbest = 4.6   bbest = 1.2   |   mmax = 5.3   bmax = −1.4   |   mmin = 3.9   bmin = 3.8
Q2 — Units
y-unit: m/s  |  x-unit: s  |  slope unit: (m/s)/s = m/s²  |  y-int unit: m/s

Part B

Q3 — Slope:
Average slope = (5.3 + 3.9) / 2 = 9.2 / 2 = 4.6 m/s²
Slope uncertainty = (5.3 − 3.9) / 2 = 1.4 / 2 = 0.7 m/s²
% slope uncertainty = (0.7 / 4.6) × 100% ≈ 15.2%
Q4 — Y-intercept:
Average b = (−1.4 + 3.8) / 2 = 2.4 / 2 = 1.2 m/s
Δb = |b_min − b_max| / 2 = |3.8 − (−1.4)| / 2 = |5.2| / 2 = 2.6 m/s
% y-int uncertainty = (2.6 / 1.2) × 100% ≈ 217% (large because b is near zero)

Part C

Q5 — Average best-fit equation:

Q7 — Minimum slope equation:
velocity(m/s) = ( 3.9 m/s² ) × time(s) + ( 3.8 m/s )

Part D

Q8: Because the maximum slope line tends to have a lower y-intercept than the minimum slope line (the lines pivot around the data in opposite ways). This means bmin > bmax and (bmin − bmax) is already positive. However, in cases where this relationship reverses, we could get a negative number. Taking the absolute value ensures the uncertainty is always expressed as a positive value, as required.
Q9: The slope unit is wrong. Slope unit = y-unit / x-unit = (m/s) / s = m/s². The student wrote m/s², which is actually correct — this would be acceleration, which makes physical sense (slope of a velocity-time graph = acceleration). So the unit m/s² is correct here; the student should double-check whether they intended m/s or m/s².
Q10: In this example it does not matter — both give 4.6 m/s² because the max and min lines are drawn symmetrically around the best-fit. However, if the data has asymmetric error bars or the min/max lines are not symmetric, the average of max and min slopes can differ from mbest. The IB standard is to always use (mmax + mmin) / 2 as the reported slope to ensure consistency with the uncertainty calculation.
