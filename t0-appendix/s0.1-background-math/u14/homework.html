Part B — Errors Found

Error 1: Column headers missing units and measurement uncertainty (should be "Mass (kg) ±___ " and "Height (m) ±0.01").
Error 2: Stat. uncertainty row 1 — student wrote 0.04, but (1.22−1.18)/2 = 0.02. Correct: 0.02
Error 3: Stat. uncertainty row 2 — (2.44−2.36)/2 = 0.04. Student wrote 0.08. Correct: 0.04
Error 4: Error bar — should be larger(0.01 meas, 0.02 stat) = 0.02 for row 1; larger(0.01, 0.04) = 0.04 for row 2. Student put 0.04 and 0.08 (doubled the stat uncertainty — wrong formula).

Part C — T/F

1. FALSE — Error bar = larger of (measurement uncertainty, statistical uncertainty), not always just statistical.
2. FALSE — The IV goes on the x (horizontal) axis. The DV goes on the y axis.
3. TRUE — If all trials are equal, Δ_stat = 0, so error bar = measurement uncertainty.
4. FALSE — Control values are listed ABOVE the table (e.g. "Control Values: ..."), not inside it.
5. TRUE — IV uncertainty → x-axis error bars; Error Bar column → y-axis error bars. ✓
