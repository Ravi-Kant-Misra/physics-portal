Q3: X error bar = ±0.001 kg (measurement uncertainty of mass). No — the Y error bars are not all the same (row 3 has 0.001, others have 0.002). You would need to use a separate error column rather than a single Fixed Value.
Q4: Δm = (0.215−0.185)/2 = 0.030/2 = ±0.015 m/kg. Final: 0.200 ± 0.015 m/kg
Q5: Δm = (3.60−2.80)/2 = 0.40 N/m. % uncertainty = 0.40/3.20 × 100 = 12.5%
Q6: Name: T² (or "Period squared") | Expression: "Period"^2
Q7: Name: KE | Expression: 0.5 * 0.5 * "v"^2 (or 0.25 * "v"^2)

Part E

Q8: The min and max lines represent the steepest and shallowest gradients that are still consistent with the data (i.e. pass through all error bars). Their difference tells us the uncertainty in the gradient. Drawing only the best-fit line would give no information about how precisely we know the slope.
Q9: T² is proportional to L (T² ∝ L), which means T ∝ √L. This is consistent with the pendulum formula T = 2π√(L/g), confirming a square-root (or equivalently, T² linear) relationship between period and length.
