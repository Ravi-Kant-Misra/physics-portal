Answer Key

Part A

Q1: X: Name=Temperature, Short=T (or θ), Units=°C, Error bar = ±0.5. Y: Name=Resistance, Short=R, Units=Ω, Error bar = ±0.3 (the error bar from data table, not just measurement uncertainty).
Q2: X error bar: Fixed Value = ±0.05 L. For Y varying error bars: create a separate column in the data table containing the error bar values for each row, then in Column Options select that column as the error bar source.

Part B

Q3: Δm = (2.70−2.20)/2 = 0.25 kg. Final: m = 2.45 ± 0.25 kg. % uncertainty = 0.25/2.45 × 100 ≈ 10.2%
Q4: Δ(gradient) = (4.30−3.74)/2 = 0.28 s²/m. Gradient = 4.02 ± 0.28 s²/m. g = 4π²/4.02 = 39.48/4.02 ≈ 9.82 m/s²

Part C

Q5: Name: 1/r² | Expression: 1/"r"^2
Q6: Name: Power (W) | Expression: "Current"^2 * "Resistance"

Part D

7. TRUE — Min slope goes from top of first error bar to bottom of last error bar. ✓
8. FALSE — Correction: Δm = (max slope − min slope) / 2 (subtraction, not addition)
9. TRUE — T² ∝ L means T² = cL, so T = √(cL) = √c × √L → T ∝ √L ✓
10. FALSE — You can draw multiple lines. Logger Pro adds best-fit lines via Analyze → Linear Fit, and you can manually draw min/max lines using drawing tools.

Part E

Q11: (a) The gradient = V/I = resistance R (Ohm's law). (b) Δm = (49.6−44.8)/2 = 2.4 Ω. (c) R = 47.2 ± 2.4 Ω. (d) The y-intercept represents voltage when current = 0. Ideally it should be 0 V (a perfect resistor with no EMF source).
Q12: Open Logger Pro → double-click X column header, enter Name/Short/Units → double-click Y column header, enter Name/Short/Units → type data into cells (points appear automatically) → double-click column header → Options tab → set Error Bar Fixed Value for X and Y → Analyze → Linear Fit for best-fit line → manually draw min and max lines through the extremes of the error bars → calculate gradient uncertainty = (max−min)/2.
