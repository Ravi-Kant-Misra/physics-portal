Formula Reference — 3-Step Method

Sin(θ) = O/H
Use: O & H involved
Rearranges to: O = H · Sin(θ) H = O / Sin(θ)
Cos(θ) = A/H
Use: A & H involved
Rearranges to: A = H · Cos(θ) H = A / Cos(θ)
Tan(θ) = O/A
Use: O & A involved
Rearranges to: O = A · Tan(θ) A = O / Tan(θ)

Answer Key

Part A — Find the Missing Side

Problem 1
O = 25 · Sin(40°) = 25 × 0.643 = 16.1
Sin(θ) = O/H → O = H·Sin(θ)
Problem 2
A = 30 · Cos(55°) = 30 × 0.574 = 17.2
Cos(θ) = A/H → A = H·Cos(θ)
Problem 3
O = 12 · Tan(35°) = 12 × 0.700 = 8.4
Tan(θ) = O/A → O = A·Tan(θ)
Problem 4
H = 9 / Sin(20°) = 9 / 0.342 = 26.3
Sin(θ) = O/H → H = O/Sin(θ)
Problem 5
H = 18 / Cos(50°) = 18 / 0.643 = 28.0
Cos(θ) = A/H → H = A/Cos(θ)
Problem 6
A = 20 / Tan(65°) = 20 / 2.145 = 9.3
Tan(θ) = O/A → A = O/Tan(θ)

Part B — Real-World Applications

Problem 7 — Ramp

A & H involved → use Cos; rearrange to H = A/Cos(θ)
