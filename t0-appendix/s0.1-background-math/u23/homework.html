Answer Key — Question 1

Ax = +20 (straight right → Ax = +A) Ay = 0
Bx = +14·cos(50°) = +14 × 0.643 = +9.0 By = −14·sin(50°) = −14 × 0.766 = −10.7 (below horizontal = negative)
Σx = 20 + 9.0 = +29.0 Σy = 0 + (−10.7) = −10.7
R = √(29.0² + 10.7²) = √(841 + 114.5) = √955.5 ≈ 30.9
θ = tan⁻¹(10.7 / 29.0) = tan⁻¹(0.369) ≈ 20° below +x axis (down-right)

Answer Key — Question 4

Ax = +25·cos(30°) = +21.7 Ay = +25·sin(30°) = +12.5
Bx = −18·cos(80°) = −3.1 (left = negative) By = +18·sin(80°) = +17.7
Cx = 0 (straight down) Cy = −10 (downward = negative)
Σx = 21.7 + (−3.1) + 0 = +18.6 Σy = 12.5 + 17.7 + (−10) = +20.2
R = √(18.6² + 20.2²) = √(346.0 + 408.0) = √754.0 ≈ 27.5 N
θ = tan⁻¹(20.2 / 18.6) = tan⁻¹(1.086) ≈ 47° above +x axis (up-right)
Σy = +20.2 (positive) → the net vertical component points upward; the resultant has a net upward vertical direction.
