IB Physics — Unit 23 — Classwork

Adding Complex Vectors

Apply the 5-step component method to each problem

Instructions: For each problem, use the 5-step method to find the resultant vector.

Step 1: Find Ax, Ay, Bx, By using trig.   Step 2: Find Σx.   Step 3: Find Σy.   Step 4: R = √(Σx² + Σy²).   Step 5: θ = tan⁻¹(Σy/Σx).

Sign convention: Right = +x  |  Left = −x  |  Up = +y  |  Down = −y

Answer Key — Problem 1

Ax = +15·cos(40°) = +11.5 Ay = +15·sin(40°) = +9.6
Bx = −10·cos(60°) = −5.0 By = +10·sin(60°) = +8.7
Σx = 11.5 + (−5.0) = +6.5 Σy = 9.6 + 8.7 = +18.3
R = √(6.5² + 18.3²) = √(42.3 + 335.0) = √377.3 ≈ 19.4
θ = tan⁻¹(18.3 / 6.5) = tan⁻¹(2.815) ≈ 70° above +x axis

Answer Key — Problem 2

Ax = 0 (straight down → no x-component) Ay = −12 (downward = negative)
Bx = −9·cos(35°) = −9 × 0.819 = −7.4 By = +9·sin(35°) = +9 × 0.574 = +5.2
Σx = 0 + (−7.4) = −7.4 Σy = −12 + 5.2 = −6.8
R = √(7.4² + 6.8²) = √(54.8 + 46.2) = √101.0 ≈ 10.1
θ = tan⁻¹(6.8 / 7.4) = tan⁻¹(0.919) ≈ 43° (down-left, below −x axis)

Answer Key — Problem 3

Ax = +8·cos(50°) = +5.1 Ay = +8·sin(50°) = +6.1
Bx = −5 (straight left) By = 0
Cx = +6·cos(25°) = +5.4 Cy = −6·sin(25°) = −2.5 (below horizontal = negative)
Σx = 5.1 + (−5) + 5.4 = +5.5 Σy = 6.1 + 0 + (−2.5) = +3.6
R = √(5.5² + 3.6²) = √(30.25 + 12.96) = √43.21 ≈ 6.6
θ = tan⁻¹(3.6 / 5.5) = tan⁻¹(0.655) ≈ 33° above +x axis
