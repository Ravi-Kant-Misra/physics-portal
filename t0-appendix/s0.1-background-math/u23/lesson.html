IB Physics — Unit 23

Adding Complex Vectors

Using X and Y Components to find Resultants

🧭

Why use components?

Problem: When two or more vectors do not point along the same line — and they are not at simple right angles to each other — you cannot simply add their magnitudes.

Solution: Break every vector into its horizontal (x) and vertical (y) parts, add those separately, then rebuild the resultant. This works for any combination of directions and any number of vectors.

This is the standard IB Physics method for adding vectors that point in arbitrary directions.

±

Sign Convention

+

Rightward components are positive

Upward components are positive

−

Leftward components are negative

Downward components are negative

Important: You must assign the correct sign before adding components. A vector pointing up-left has a negative x-component and a positive y-component.

📐

The 5-Step Method

1
Find all x- and y-components

Use trigonometry to decompose each vector into its horizontal and vertical parts. Apply the sign convention.

Ax = A·cos(θ)  |  Ay = A·sin(θ)
2
Sum all x-components (Σx)

Add together every x-component, keeping their signs. The result is the x-component of the resultant.

Σx = Ax + Bx (= Rx)
3
Sum all y-components (Σy)

Add together every y-component, keeping their signs. The result is the y-component of the resultant.

Σy = Ay + By (= Ry)
4
Find resultant magnitude (R)

Use Pythagoras on the component sums. The resultant magnitude is always positive.

R = √(Σx² + Σy²)
5
Find resultant direction (θ)

Use the inverse tangent of the y-sum over the x-sum. State the direction clearly relative to a reference axis.

θ = tan⁻¹(Σy / Σx)
📊

Component Table Template

Always organise your work in this table format:

Vectorx-componenty-component
AAxAy
BBxBy
ΣΣx = Ax + BxΣy = Ay + By
R = √(Σx² + Σy²)
θ = tan⁻¹(Σy / Σx)

For more than two vectors, simply add more rows — one per vector. The sums in the Σ row always give you Rx and Ry.

Test your understanding 3 questions

1. Vector A has magnitude 12 and points 60° above horizontal to the RIGHT. Which expression gives Ax?

Correct! A points right, so Ax is positive. The x-component always uses cos(θ) where θ is measured from horizontal. Ax = +12·cos(60°) = +6.0
Not quite. A points rightward, so Ax must be positive. Also, horizontal components use cos, not sin.
Careful — horizontal components use cos(θ), not sin(θ). Sin gives the vertical component.
Two errors: A points right (so Ax is positive), and horizontal components use cos(θ).

2. After finding components, you get Σx = −5 and Σy = +12. What is the resultant magnitude R?

Correct! R = √((−5)² + 12²) = √(25 + 144) = √169 = 13
Check the formula: R = √(Σx² + Σy²). Don't subtract — always add the squares.
Don't add the magnitudes directly — use Pythagoras: R = √(5² + 12²).
You found Σx² + Σy² = 169, but R is the square root of that value.

3. A vector points straight downward with magnitude 8. What are its x and y components?

Correct! Straight down means all the magnitude is in the y-direction, and downward is negative: x = 0, y = −8.
A downward vector has no horizontal component (x = 0). It also points in the negative y direction.
Close — the vector is purely vertical (x = 0) — but downward is the negative y-direction.
A downward vector has no horizontal component. x = 0, and since it points down, y = −8.
