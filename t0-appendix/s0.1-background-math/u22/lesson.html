Finding X & Y Components

Using Trigonometry to Decompose Vectors into Horizontal and Vertical Parts
📋

The Core Idea

Every vector that is not purely horizontal or vertical forms a right triangle with its x and y components. The vector itself is the hypotenuse, the x-component is the horizontal leg, and the y-component is the vertical leg.

Because we always have a right triangle, we can use trigonometry (SOH-CAH-TOA) to find the exact values of each component from the vector's magnitude and direction angle.

Angle from Horizontal (Standard)

θ is measured from the horizontal. x-component is adjacent (cos), y-component is opposite (sin).

Angle from Vertical (Alternate)

θ is measured from the vertical. Now x-component uses sin and y-component uses cos — they swap!

📝

The Formulas

Case 1 — Angle Measured from Horizontal

X-Component (horizontal)
x = H · Cos(θ)
Adjacent side to the angle → cosine
Y-Component (vertical)
y = H · Sin(θ)
Opposite side to the angle → sine
H = magnitude (hypotenuse)  |  θ = angle measured from the horizontal axis

Case 2 — Angle Measured from Vertical

X-Component (horizontal)
x = H · Sin(θ)
Opposite side to the angle → sine
Y-Component (vertical)
y = H · Cos(θ)
Adjacent side to the angle → cosine
When the angle is from vertical, sin and cos swap! The x and y roles reverse.

The Golden Rule

Always identify which axis the angle is measured FROM before applying the formula.

⚠️ Common Mistake

Students often apply Cos for x and Sin for y without checking the angle reference. If the given angle is from the vertical, you must swap the trig functions. The physical components don't change — only which formula you use changes.

💡 Memory Tip: SOH-CAH-TOA Check

Ask yourself: "Is the x-component (horizontal leg) adjacent to or opposite from the given angle?" If adjacent → use Cos. If opposite → use Sin. This always gives the right answer regardless of which axis the angle is referenced from.

🔎

Worked Examples — Angle from Horizontal

Case 1

θ measured from the horizontal axis

Example 1 θ = 30° from horizontal
A = 20 θ = 30° from horizontal
X-Component
Ax = A · Cos(θ)
Ax = 20 · Cos(30°)
Ax = 17.3
Y-Component
Ay = A · Sin(θ)
Ay = 20 · Sin(30°)
Ay = 10.0
Cos(30°) ≈ 0.866, Sin(30°) = 0.500  |  Check: √(17.3² + 10²) ≈ 20 ✓
Example 2 θ = 60° from horizontal
B = 15 θ = 60° from horizontal
X-Component
Bx = 15 · Cos(60°)
Bx = 7.5
Y-Component
By = 15 · Sin(60°)
By = 13.0
Notice: larger angle from horizontal → larger y-component and smaller x-component. Makes physical sense!
Example 3 θ = 45° from horizontal
C = 6 θ = 45° from horizontal
X-Component
Cx = 6 · Cos(45°)
Cx = 4.2
Y-Component
Cy = 6 · Sin(45°)
Cy = 4.2
At 45° the components are equal because sin(45°) = cos(45°) ≈ 0.707. The vector splits evenly between x and y.
Example 4 θ = 20° from horizontal
D = 10 θ = 20° from horizontal
X-Component
Dx = 10 · Cos(20°)
Dx = 9.4
Y-Component
Dy = 10 · Sin(20°)
Dy = 3.4
Small angle from horizontal → x-component dominates. The vector is nearly horizontal.
🔎

Worked Examples — Angle from Vertical

Case 2

θ measured from the vertical axis

💡 Key Insight

The same physical vector has the same components regardless of how the angle is stated. When the angle is measured from vertical, it is the complement of the horizontal angle (they add to 90°). So sin and cos swap — but the final component values stay the same!

Example 1 θ = 60° from vertical (same vector as before: A=20)
A = 20 θ = 60° from vertical
X-Component
Ax = A · Sin(θ)
Ax = 20 · Sin(60°)
Ax = 17.3
Y-Component
Ay = A · Cos(θ)
Ay = 20 · Cos(60°)
Ay = 10.0
Same answers as Example 1 (Case 1)! 30° from horizontal = 60° from vertical. The angle flips, the functions swap, the components are identical.
Example 2 θ = 30° from vertical (same vector as before: B=15)
B = 15 θ = 30° from vertical
X-Component
Bx = 15 · Sin(30°)
Bx = 7.5
Y-Component
By = 15 · Cos(30°)
By = 13.0
Same components as B in Case 1. Angle 60° horizontal = 30° vertical. Functions swap, answers match.
Example 3 θ = 45° from vertical
C = 6 θ = 45° from vertical
X-Component
Cx = 6 · Sin(45°)
Cx = 4.2
Y-Component
Cy = 6 · Cos(45°)
Cy = 4.2
45° from vertical = 45° from horizontal. Components are still equal because 45° is its own complement.
Example 4 θ = 70° from vertical
D = 10 θ = 70° from vertical
X-Component
Dx = 10 · Sin(70°)
Dx = 9.4
Y-Component
Dy = 10 · Cos(70°)
Dy = 3.4
70° from vertical = 20° from horizontal. Same Dx and Dy as Case 1 Example 4.

Side-by-Side Comparison

Vector Angle (from horizontal) Angle (from vertical) x-component y-component
A = 20 30° → x = 20Cos(30°) 60° → x = 20Sin(60°) 17.3 10.0
B = 15 60° → x = 15Cos(60°) 30° → x = 15Sin(30°) 7.5 13.0
C = 6 45° → x = 6Cos(45°) 45° → x = 6Sin(45°) 4.2 4.2
D = 10 20° → x = 10Cos(20°) 70° → x = 10Sin(70°) 9.4 3.4

Pattern: The horizontal angle and vertical angle always add up to 90° (they are complementary). When you switch from one reference to the other, sin and cos swap — but the actual component values remain identical.

This makes sense: the physical vector doesn't change just because you described its angle differently!
