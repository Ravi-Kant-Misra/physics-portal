Answer Key

Part A — Angle from Horizontal

Problem 1   H=18, θ=40°
x = 18·Cos(40°) = 13.8
y = 18·Sin(40°) = 11.6
Check: √(13.8²+11.6²) ≈ 18 ✓
Problem 2   H=25, θ=55°
x = 25·Cos(55°) = 14.3
y = 25·Sin(55°) = 20.5
Large angle → y > x
Problem 3   H=30, θ=25°
x = 30·Cos(25°) = 27.2
y = 30·Sin(25°) = 12.7
Small angle → x dominates
Problem 4   H=12, θ=75°
x = 12·Cos(75°) = 3.1
y = 12·Sin(75°) = 11.6
Near-vertical vector: y ≈ H

Part B — Angle from Vertical

Problem 5   H=20, θ=35° from vert.
x = 20·Sin(35°) = 11.5
y = 20·Cos(35°) = 16.4
Equiv. to 55° from horizontal
Problem 6   H=14, θ=50° from vert.
x = 14·Sin(50°) = 10.7
y = 14·Cos(50°) = 9.0
Equiv. to 40° from horizontal
Problem 7   H=9, θ=20° from vert.
x = 9·Sin(20°) = 3.1
y = 9·Cos(20°) = 8.5
Equiv. to 70° from horizontal
Problem 8   H=32, θ=65° from vert.
x = 32·Sin(65°) = 29.0
y = 32·Cos(65°) = 13.5
Equiv. to 25° from horizontal

Part C — Mixed Problems

Problem 9: vx = 22·Cos(35°) = 18.0 m/s, vy = 22·Sin(35°) = 12.6 m/s
Problem 10: Fx = 50·Sin(40°) = 32.1 N, Fy = 50·Cos(40°) = 38.3 N (angle from vertical, so sin and cos swap)
Problem 11: Student A: x = 16·Cos(30°) = 13.9, y = 16·Sin(30°) = 8.0  |  Student B: x = 16·Sin(60°) = 13.9, y = 16·Cos(60°) = 8.0  → Identical!

Part D — Conceptual

Q12: As θ approaches 90° from horizontal, Cos(θ) → 0 so the x-component shrinks to zero. Sin(θ) → 1 so the y-component approaches H. The vector becomes more and more vertical.
Q13: The student is correct when angles are always measured from the horizontal. They will get wrong answers when an angle is measured from the vertical — in that case x uses Sin(θ) and y uses Cos(θ).
