Part B — Applied Problems

Problem 7: vx = 18·Cos(42°) = 13.4 m/s  |  vy = 18·Sin(42°) = 12.0 m/s
Problem 8: Angle from vertical → Fx = 75·Sin(28°) = 35.2 N  |  Fy = 75·Cos(28°) = 66.2 N
Problem 9: Method 1 (15° from vert.): vx = 250·Sin(15°) = 64.7 m/s, vy = 250·Cos(15°) = 241.5 m/s  |  Method 2 (75° from horiz.): vx = 250·Cos(75°) = 64.7 m/s, vy = 250·Sin(75°) = 241.5 m/s  → Results match ✓

Part C — Error Analysis

Problem 10: Mistake: The angle (55°) is from the vertical, so x needs Sin and y needs Cos — the student used the horizontal-angle formulas instead.  |  Correct: x = 40·Sin(55°) = 32.8, y = 40·Cos(55°) = 22.9
Problem 11: Mistake: The angle (30°) is from the horizontal, so x needs Cos and y needs Sin — the student swapped them, applying the vertical-angle formulas.  |  Correct: x = 10·Cos(30°) = 8.7, y = 10·Sin(30°) = 5.0

Part D — Extension

Problem 12: (a) H = √(8² + 6²) = √100 = 10  |  (b) From horizontal: θ = arctan(6/8) = arctan(0.75) = 36.9°  |  (c) From vertical: 90° − 36.9° = 53.1°
Problem 13: At 30°: x = 20·Cos(30°) = 17.3  |  At 60°: x = 20·Cos(60°) = 10.0  |  17.3 ÷ 2 = 8.65 ≠ 10.0. The student is incorrect. Doubling the angle does not halve the x-component because cosine is not a linear function (Cos(2θ) ≠ Cos(θ)/2).
