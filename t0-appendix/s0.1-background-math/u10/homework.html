Formula Reference

Multiply or divide (y = a×b or a÷b):
Δy/y = Δa/a + Δb/b
Δy = (Δa/a + Δb/b) × y
Three steps every time:
1. Compute result y normally
2. Find fractional Δ for each factor, add them
3. Multiply fractional × y → absolute Δy
Percent uncertainties:
Just add them directly.
%_result = %₁ + %₂
Absolute to fractional:
fractional = Δa / a
percent = (Δa / a) × 100%
REMEMBER: Adding/Subtracting → add ABSOLUTE uncertainties  |  Multiplying/Dividing → add FRACTIONAL uncertainties
Always add fractional (never subtract) Works the same for both × and ÷ Common mistake: adding absolute uncertainties for ×÷ problems

Part A — Direct Calculation

Problem 1
(6±3 m) × (4±1 m)
24 m² ± 0.75 (frac) → 24 ± 18 m²
y = 24. Δa/a = 3/6 = 0.50. Δb/b = 1/4 = 0.25. Total = 0.75. Absolute: 0.75 × 24 = 18 m².
Problem 2
(5.0±1.0 kg) × (3.0±0.3 m/s²)
15 N ± 0.30 (frac) → 15 ± 4.5 N
y = 15. Δa/a = 1.0/5.0 = 0.20. Δb/b = 0.3/3.0 = 0.10. Total = 0.30. Absolute: 0.30 × 15 = 4.5 N.
Problem 3
(80 J ±5%) ÷ (20 s ±10%)
4.0 W ± 15%
y = 4.0. Add percents: 5% + 10% = 15%. Absolute: 0.15 × 4.0 = 0.6 W. So 4.0 ± 0.6 W.
Problem 4
(18±3 V) ÷ (6.0±1.5 A)
3.0 Ω ± 0.42 (frac) → 3.0 ± 1.25 Ω
y = 3.0. Δa/a = 3/18 = 0.167. Δb/b = 1.5/6.0 = 0.25. Total = 0.417. Absolute: 0.42 × 3.0 ≈ 1.25 Ω.

Problem 5

y = 30. Frac 1: 0.75/7.5 = 0.10. Frac 2: 25% = 0.25. Total = 0.35. Absolute: 0.35 × 30 = 10.5.

Part C — Real-World Applications

Q9 (Sail Area): Area = ½ × 4.2 × 6.0 = 12.6 m². Frac: 0.3/4.2 = 0.071, 0.6/6.0 = 0.10. Total = 0.171 ≈ 0.17. Absolute: 0.17 × 12.6 ≈ 2.1 m². Answer: 12.6 ± 2.1 m². (The constant ½ contributes zero uncertainty.)
Q10 (Density): Density = 150/125 = 1.20 g/mL. Frac: 5/150 = 0.033, 10/125 = 0.080. Total = 0.113. Absolute: 0.113 × 1.20 ≈ 0.136 ≈ 0.1 g/mL. Answer: 1.20 ± 0.14 g/mL. The known value 1.25 g/mL is within 1.25 − 1.20 = 0.05 g/mL of the result, and 0.05 < 0.14, so YES — the results agree within uncertainty.
Q11 (Ohm's Law): R = 12.0/2.4 = 5.0 Ω. Frac: voltage = 0.6/12.0 = 0.05 (5%); current = 5% = 0.05. Total = 0.10 (10%). Absolute: 0.10 × 5.0 = 0.5 Ω. Answer: 5.0 ± 0.5 Ω. Both measurements contribute equally (each 5%). To reduce total uncertainty, improve either one — they both have the same impact.

Part D — Error Analysis

Q12 Error: Mistake: student added absolute uncertainties (2 + 0.6 = 2.6) instead of fractional. Correct: Frac 1 = 2/8 = 0.25. Frac 2 = 0.6/3 = 0.20. Total = 0.45. Absolute: 0.45 × 24 = 10.8 m². Correct answer: 24 ± 11 m².
Q13 Error: Mistake: student subtracted the percentage uncertainties, reasoning that division means subtract. This is WRONG. For multiplication AND division, uncertainties always ADD. Correct: 10% + 25% = 35%. Answer: 5 A ± 35%, or 5 ± 1.75 A.

Part E — Short Answer

Q14: When adding/subtracting, uncertainties represent absolute ranges that shift the result by a fixed amount, so they stack directly. When multiplying/dividing, what matters is proportionality — a 10% error in a large number contributes the same relative effect to the result as a 10% error in a small number. Fractional uncertainty captures what fraction of the measurement the uncertainty is, and those fractions combine to give the fraction of error in the result.
Q15: Current combined uncertainty = 5% + 2% = 7%. If you halve the scale (5% → 2.5%): new total = 2.5% + 2% = 4.5% (improvement of 2.5%). If you halve the ruler (2% → 1%): new total = 5% + 1% = 6% (improvement of only 1%). Upgrade the scale — it contributes more (5%) and halving it saves 2.5% vs the ruler saving only 1%.
Q16 (Challenge): KE = ½ × 2.0 × (5.0)² = ½ × 2.0 × 25 = 25 J. Frac for m: 0.2/2.0 = 0.10. Frac for v²: 2 × (0.5/5.0) = 2 × 0.10 = 0.20. Total frac = 0.10 + 0.20 = 0.30. Absolute: 0.30 × 25 = 7.5 J. Answer: KE = 25 ± 7.5 J (30% uncertainty). Notice the squared velocity contributes double because squaring amplifies relative errors.
