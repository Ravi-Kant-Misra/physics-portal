Part C

Q10: (a) a = (101.4+98.6)/2 = 100.0°C; Δa = (101.4−98.6)/2 = 1.4°C → 100.0 ± 1.4°C. (b) No — 98.5°C is below the minimum (98.6°C), so it is not a possible value.
Q11: Student A: 15±4 → range 11 to 19 cm. Student B: 12 to 22 cm. There is overlap (12–19 cm), so their measurements are partially consistent. However, Student B's range is wider. The measurements do not fully agree — Student B allows values up to 22 cm which Student A does not.
