Classwork — Position-Time Graphs

Practice reading and calculating from position-time graphs.

Reference — Key Formulas and Rules

Velocity from Graph
v = Rise / Run = Δd / Δt
Y-intercept
= Starting position (P₀)
Displacement
Δd = P_final − P_initial
Distance
Sum of all |segment changes|
Horizontal line
Object is stationary (v = 0)
Negative slope
Moving backward (v < 0)

Answer Key — Classwork: Position-Time Graphs

A1
a) Starting position = 1m (y-intercept)
b) Velocity = +1.0 m/s
v = Rise/Run = (5m − 1m)/(4s − 0s) = 4m/4s = 1.0 m/s
A2
a) Starting position = 6m
b) Velocity = −0.5 m/s (moving backward)
v = Rise/Run = (2m − 6m)/(8s − 0s) = −4m/8s = −0.5 m/s
A3
a) Starting position = 5m
b) Velocity = 0 m/s (horizontal line = stationary)
v = 0m/10s = 0 m/s. A horizontal line always means zero velocity.
A4
a) Starting position = 0m
b) Velocity = +0.5 m/s
v = (3m − 0m)/(6s − 0s) = 3m/6s = 0.5 m/s
A5
a) Starting position = 3m
b) Velocity = +0.5 m/s
v = (8m − 3m)/(10s − 0s) = 5m/10s = 0.5 m/s
A6
a) Starting position = 7m
b) Velocity = −1.0 m/s
v = (3m − 7m)/(4s − 0s) = −4m/4s = −1.0 m/s
B1
Starting position = 2m
Seg 1 slope: v = (6m−2m)/(4s−0s) = +4m/4s = +1.0 m/s → velocity at t=2s = +1.0 m/s
Seg 2 slope: v = 0m/5s = 0 m/s → velocity at t=7s = 0 m/s
Seg 3 slope: v = (3m−6m)/(12s−9s) = −3m/3s = −1.0 m/s → velocity at t=11s = −1.0 m/s
B2
Starting position = 5m
Seg 1 slope: v = 0m/3s = 0 m/s → velocity at t=1s = 0 m/s
Seg 2 slope: v = (1m−5m)/(7s−3s) = −4m/4s = −1.0 m/s → velocity at t=5s = −1.0 m/s
Seg 3 slope: v = (6m−1m)/(12s−7s) = +5m/5s = +1.0 m/s → velocity at t=10s = +1.0 m/s
B3
Total Distance = 10m  |  Total Displacement = −1m
Seg 1: |7m − 4m| = 3m (upward)
Seg 2: |7m − 7m| = 0m (flat)
Seg 3: |3m − 7m| = 4m (downward, magnitude only)
Distance = 3 + 0 + 4 = 7m ... Wait — let me recompute: Seg1: 7−4=3m, Seg3: 3−7=−4m, |−4|=4m
Distance = 3+0+4 = 7m → Displacement = 3m − 4m = end−start: P_final=3m, P_initial=4m → Δd=3−4=−1m
Total Distance = 3m + 0m + 4m = 7m
B4 (first 9 seconds only: segments through t=9)
Distance after 9s = 9m  |  Displacement after 9s = +3m
Seg 1 (0→3s): |7m−1m|=6m
Seg 2 (3→6s): |7m−7m|=0m
Seg 3 (6→9s): |4m−7m|=3m
Distance = 6+0+3 = 9m
Displacement = P(9s)−P(0s) = 4m−1m = +3m
C1
Positive slope: object is moving in the positive (forward) direction at constant speed.
Zero slope (horizontal line): object is stationary — its position is not changing.
Negative slope: object is moving in the negative (backward) direction at constant speed.
C2
a) Distance = 10m  |  b) Displacement = −2m
c) Yes. The negative displacement means the object ended up 2m behind where it started, so the net movement was in the backward direction.
Forward: |7m−3m|=4m; Backward: |1m−7m|=6m; Distance=4+6=10m
Displacement=1m−3m=−2m
C3
Object B is faster (|−3 m/s| > |2 m/s|, so speed=3 m/s vs 2 m/s).
Object B is moving backward (negative velocity = negative direction).
The negative sign means Object B is moving in the direction of decreasing position — the same road but in reverse.
