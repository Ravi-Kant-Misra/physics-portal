Position-Time Graphs

Learn how to read, draw, and interpret position-time graphs to analyze complex motion.

Definition

What is a Position-Time Graph?

Position-time graphs show an object's position on the y-axis and the time on the x-axis. They are used to show complicated motion on a simple picture.

Y-Axis
Position (P)
Unit: metres (m)
X-Axis
Time (t)
Unit: seconds (s)

A blank P-T graph — position on y-axis, time on x-axis.

Reading the Graph

How to Read a Position-Time Graph

The graph tracks where an object is at each moment in time. Each point on the graph tells you two things: the time (x-value) and the position (y-value) at that moment.

Example — A Full Journey

Watch how the graph builds up as an object moves:

①t = 0 → 3sP = 3m → 6mMoving forward
②t = 3 → 6sP = 6m (constant)Stationary
③t = 6 → 8sP = 6m → 2mMoving backward
④t = 8 → 12sP = 2m (constant)Stationary
Interpreting Slope

What the Slope Tells You

SlopeGraph ShapeMeaning
PositiveLine goes up ↗Object moving forward (positive velocity)
ZeroHorizontal line →Object is stationary (v = 0 m/s)
NegativeLine goes down ↘Object moving backward (negative velocity)
SteeperMore verticalFaster speed
ShallowerMore horizontalSlower speed
Remember

A horizontal line on a P-T graph does NOT mean the object is at zero — it means the object is not moving (its position is constant).

Two Key Patterns

Patterns in the Graph

Pattern 1 — Y-Intercept = Starting Position

Key Insight

The y-intercept of the position-time graph is the starting position of the object. It tells you where the object was at t = 0.

Pattern 2 — Slope = Velocity

Key Insight

The slope of a P-T graph equals the velocity of the object. A steeper slope = higher velocity. A negative slope = moving backward.

Slope = Rise / Run = Displacement / Time = Velocity

Rise = Change in position (displacement)  |  Run = Change in time

Worked Examples

Example 1 — Starting Position and Velocity

Example 1

A straight line goes from (t=0, P=2m) to (t=12s, P=5m).

Q: What is the starting position?
Answer: 2m — The y-intercept is 2m, so the object started at position 2m.
Q: What is the velocity?
Answer: 0.25 m/s
v = Rise/Run = (5m − 2m) / (12s − 0s) = 3m / 12s = 0.25 m/s

Example 2 — Multi-Segment Velocity

Example 2
Q: What is the starting position?
Answer: 4m — y-intercept = 4m.
Q: Velocity at t = 2s? (on segment 1)
+0.5 m/s   →   v = +2m / 4s = +0.5 m/s
Q: Velocity at t = 6s? (on segment 2)
0 m/s   →   v = 0m / 4s = 0 m/s (horizontal line)
Q: Velocity at t = 10s? (on segment 3)
−1.5 m/s   →   v = −6m / 4s = −1.5 m/s (moving backward)
Distance vs Displacement

Finding Distance and Displacement from a P-T Graph

Rule 1 — Distance

To find the distance traveled over a time: add up the distance traveled in each line segment (always use the magnitude — no negatives).

Rule 2 — Displacement

To find the displacement traveled over a time: compare the beginning and end point of the motion, and ignore the rest of the graph.
Displacement = Pfinal − Pinitial

Worked Example — Distance & Displacement

Graph: starts at P=2m, rises to P=6m, flat, falls to P=0m.

Distance Traveled:
Segment 1: |6m − 2m| = 4m
Segment 2: |6m − 6m| = 0m
Segment 3: |0m − 6m| = 6m
Total Distance = 4m + 0m + 6m = 10m
Displacement:
Δd = Pfinal − Pinitial = 0m − 2m = −2m

Example — Distance and Displacement after 7 Seconds

Example 4
Q: What is the distance traveled after 7 seconds?
Segment 1: |7m − 2m| = 6m (Note: approximated as 6m from graph)
Segment 2: |7m − 7m| = 0m
Segment 3: |2m − 7m| = 5m
Total Distance = 6m + 0m + 5m = 11m
Q: What is the displacement after 7 seconds?
Δd = P(7s) − P(0s) = 2m − 1m = +1m

The object ends at the same relative position it started — the net displacement is +1m.

Summary

Key Takeaways

Y-intercept
= Starting position of the object
Slope
= Velocity of the object (Rise ÷ Run)
Horizontal line
= Object is stationary (v = 0)
Negative slope
= Moving backward (negative velocity)
Distance
Sum of all segment lengths (always +)
Displacement
Pfinal − Pinitial (can be negative)
v = Δd / Δt = Rise / Run

The slope formula links velocity, displacement, and time.
