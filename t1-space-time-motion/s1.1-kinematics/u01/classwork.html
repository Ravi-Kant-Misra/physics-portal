Answer Key

Part A — Path Problems

Problem 1
d = 8 m, s = 2 m east
5+3=8m total; ends 5−3=2m east of start
Problem 2
d = 20 m, s = 0 m
10+10=20m; returns to start, so s=0
Problem 3
d = 14 m, s = 10 m
6+8=14m; √(6²+8²)=√100=10m
Problem 4
d = 12 m, s = 4 m right
4+4+4=12m; net = 4+4−4=4m right

d = 8 m, s = 0 m

d = 17 m, s = 13 m
5+12=17m; √(5²+12²)=√169=13m

Part C — Short Answer

11. No. Displacement can never be greater than distance. The straight-line path is always the shortest possible route between two points — the actual path must be equal to or longer. They are equal only when the path is a straight line with no backtracking.
12. (a) Distance = 42 km (total path covered). (b) Displacement = 0 m (they finish at the same place they started, so x_f = x_i).
13. The negative sign tells you the object ended up in the negative direction relative to where it started (e.g. to the left if positive is right, or downward if positive is up). The magnitude (8m) tells you how far from start.
