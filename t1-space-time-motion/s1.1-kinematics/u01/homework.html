Part A — Calculations

Problem 1
d = 11 m, s = 5 m west
8+3=11m; net = 8−3=5m, still westward
Problem 2
d = 60 m, s = 0 m
15×4=60m; closed square → back to start
Problem 3
d = 21 m, |s| = 15 m
9+12=21m; √(9²+12²)=√225=15m
Problem 4
d = 13 m, s = −3 m
path: +5m then −8m = 13m; s = 1−4 = −3m
Problem 5
d = 8 m, s = +8 m
s = 3−(−5) = +8m; straight path → d = s
Problem 6
d = 31 m, |s| = 25 m
7+24=31m; √(7²+24²)=√(49+576)=√625=25m

Part B — True or False

FALSE
Distance is a scalar (no direction). It is not a vector.
TRUE
Displacement = x_f − x_i. If start = end, then s = 0 regardless of path.
FALSE
Displacement can never be larger than distance. Speed is irrelevant. The straight-line is always the shortest path.
TRUE
