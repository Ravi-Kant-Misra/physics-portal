Distance & Displacement

Understanding the difference between how far you travel and where you end up

📖

Definitions

Distance

DefinitionThe total scalar length of the actual path an object moved.
Variabled
Unitm (metres)
Scalar — magnitude only

Displacement

DefinitionA vector that points from where the object started to where it ended, regardless of the path it took.
Variables
Unitm (metres)
Vector — magnitude + direction

Key Distinction

Distance adds up every metre you walk — it only grows. Displacement is a straight-line arrow from start to finish — it depends on direction, and can be zero even if you walked a long way.

Working on a Number Line

The Rule for 1D Motion

When an object moves along an axis (a straight line), we must define:
1. A zero point (origin) — where we measure everything from
2. A positive direction (usually right or up) — the opposite direction is negative

Displacement = Final position − Initial position    s = x_f − x_i

💡

Key Insights

Remember these rules

Distance is always ≥ 0
You can never travel a negative distance. It is always a positive number (or zero).
Displacement can be negative
A negative displacement means the object ended up in the negative direction from where it started.
Displacement can be zero
If the object returns to its starting point, displacement = 0 — even if distance was huge.
|Displacement| ≤ Distance
The straight-line distance is always ≤ the total path. They're equal only when the path is a straight line with no backtracking.
