Speed & Velocity

Understanding how fast objects move — and the crucial difference direction makes

📖

Definitions

Speed

DefinitionSpeed is the distance an object travels divided by the time it takes to travel that distance. It is a scalar.
VariableSpeed
Unitms⁻¹
FormulaSpeed = d/t
Scalar — magnitude only

Velocity

DefinitionVelocity is the displacement an object travels divided by the time it takes to travel that displacement. It is a vector.
VariableV
Unitms⁻¹
FormulaV = s/t
Vector — magnitude + direction

Variable Key

d
Distance — total path length (scalar)
s
Displacement — straight-line with direction (vector)
t
Time — measured in seconds

A quick note on the unit:

ms⁻¹ = m/s

Both forms mean exactly the same thing: metres per second.

8m ÷ 2s = 4 m/s = 4 ms⁻¹

An object that travels 8 metres in 2 seconds has a speed of 4 ms⁻¹

📐

Worked Examples

Example 1 — L-shaped path (3 segments, time = 3 s)

The stick figure walks 3m right, then 3m up, then 3m right again.

Key Insight

Because the path is not straight, distance (9m) ≠ displacement (3m). Therefore speed (3 ms⁻¹) ≠ velocity (1 ms⁻¹). They are different quantities!

Example 2 — Straight-line path (time = 2 s)

The stick figure walks 6m to the right in a straight line from Start to End.

Distance (d)
6 m
Displacement (s)
6 m right
Speed
3 ms⁻¹
6m ÷ 2s
Velocity
3 ms⁻¹ right
6m ÷ 2s

Key Insight

When the path is a straight line in one direction, distance = displacement, so speed = |velocity|. Both are 3 ms⁻¹ here.

Example 3 — Out-and-back path (time = 4 s)

The stick figure walks 6m right, then 6m left, returning to the starting point.

Distance (d)
12 m
6 + 6
Displacement (s)
0 m
Start = End position
Speed
3 ms⁻¹
12m ÷ 4s
Velocity
0 ms⁻¹
0m ÷ 4s

Key Insight — The most powerful distinction!

The object was definitely moving (speed = 3 ms⁻¹), but because it returned to where it started, displacement = 0, so velocity = 0. Speed and velocity can be very different numbers!

📏

Important Note: Negative Velocity

Important Note:

If velocity is negative, it means the object is traveling in the direction we've defined as negative.

We define a positive direction (usually rightward) and anything moving the other way gets a negative sign.

Negative Velocity — What it means

A velocity of −7 ms⁻¹ means: the object is moving at 7 ms⁻¹ in whichever direction was defined as negative. The speed is still 7 ms⁻¹ (magnitude only, never negative).

Key Rules

Remember these rules

Speed uses distance; velocity uses displacement
Speed = d/t (scalar). Velocity = s/t (vector). Different inputs, different outputs.
Speed is always ≥ 0; velocity can be negative
Speed only has magnitude. Velocity has direction — negative velocity means motion in the negative direction.
Straight-line, one direction: speed = |velocity|
When distance = displacement, speed and the magnitude of velocity are equal (Example 2).
Returning to start: displacement = 0, velocity = 0
Even if the object was moving fast, finishing where you started means velocity = 0 (Example 3). Speed can still be non-zero.
Same unit for both: ms⁻¹
Both speed and velocity are measured in ms⁻¹ (= m/s). The difference is whether direction is included.
Comparison table is your anchor
When in doubt, ask: am I using distance or displacement? Am I including direction? That determines whether it's speed or velocity.

Quick Check

Test Your Understanding

Click the correct answer for each question.

1. A runner completes a 400m lap of a track in 50 s, finishing exactly where they started. What is their velocity?
Correct! Displacement = 0 (they finish where they started), so velocity = 0/50 = 0 ms⁻¹. Speed = 400/50 = 8 ms⁻¹, but velocity is 0.
Not quite. They finish where they started, so displacement = 0. Velocity = s/t = 0/50 = 0 ms⁻¹. (Speed = 400/50 = 8 ms⁻¹ — different!)
2. Which formula correctly gives velocity?
Correct! Velocity = s/t, where s is displacement. Speed = d/t, where d is distance. The key difference is displacement vs distance.
Velocity = s/t (using displacement s). Speed = d/t (using distance d). Remember: velocity uses displacement!
3. An object has a velocity of −5 ms⁻¹. What does the negative sign tell you?
Correct! Negative velocity means motion in the direction defined as negative (e.g. leftward if right = positive). The speed is still +5 ms⁻¹ — speed is never negative.
Negative velocity means the object is traveling in the negative direction. The speed (magnitude) is still 5 ms⁻¹ — speed is always positive or zero.
4. An object walks 3m right, then 3m up, in 3 s. What is its speed?
Correct! Distance = 3+3 = 6m. Speed = d/t = 6/3 = 3 ms⁻¹. (Velocity would use displacement = √(3²+3²) ≈ 4.24m, giving V ≈ 1.41 ms⁻¹.)
Speed = d/t = (3+3)/3 = 6/3 = 3 ms⁻¹. Total distance is 6m (path length), not the displacement. Speed uses distance.
