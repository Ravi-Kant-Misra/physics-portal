Reference Formulas

Speed = d/t — uses distance, scalar (ms⁻¹)
Velocity (V) = s/t — uses displacement, vector (ms⁻¹)
d = total path length (always ≥ 0)  |  s = displacement (can be negative)
ms⁻¹ = m/s — unit is the same for both speed and velocity
Study tip: Always ask yourself — am I using distance or displacement? Is this a scalar or a vector? Show full working for every calculation.

Answer Key

Part A — Calculations

Problem 1 — 12m east, 4s
Speed = 3 ms⁻¹  |  Velocity = 3 ms⁻¹ east
d = 12m, s = 12m east (straight line). Speed = 12/4 = 3 ms⁻¹. Velocity = 12/4 = 3 ms⁻¹ east. Since the path is straight and in one direction, speed = |velocity|.
Problem 2 — 18m right then 6m left, 6s
Speed = 4 ms⁻¹  |  Velocity = 2 ms⁻¹ right
d = 18+6 = 24m. s = 18−6 = 12m right. Speed = 24/6 = 4 ms⁻¹. Velocity = 12/6 = 2 ms⁻¹ right. Note: speed ≠ |velocity| because the path changed direction.
Problem 3 — Square loop back to start, 20s
Speed = 1 ms⁻¹  |  Velocity = 0 ms⁻¹
d = 5+5+5+5 = 20m. s = 0m (returns to start: N+S cancel, E+W cancel → displacement = 0). Speed = 20/20 = 1 ms⁻¹. Velocity = 0/20 = 0 ms⁻¹. Classic example of speed ≠ velocity.
Problem 4 — +14m to +2m, 4s
Speed = 3 ms⁻¹  |  Velocity = −3 ms⁻¹
d = |14−2| = 12m. s = 2−14 = −12m (moving leftward = negative). Speed = 12/4 = 3 ms⁻¹. Velocity = −12/4 = −3 ms⁻¹. Negative velocity confirms leftward motion.
Problem 5 — velocity = −9 ms⁻¹ for 3s (rearranging)
Distance = 27m  |  Displacement = −27m
Rearranging V = s/t: s = V × t = −9 × 3 = −27m. Displacement = −27m (in the negative direction). Distance = |−27| = 27m (always positive). Speed = 27/3 = 9 ms⁻¹.
Problem 6 — Object A vs Object B
A: 3 ms⁻¹, 3 ms⁻¹ right  |  B: 3 ms⁻¹, 0 ms⁻¹
Object A: d=6m, s=6m right, t=2s. Speed = 6/2 = 3 ms⁻¹. V = 6/2 = 3 ms⁻¹ right. Object B: d=12m (6+6), s=0m (returns to start), t=4s. Speed = 12/4 = 3 ms⁻¹. V = 0/4 = 0 ms⁻¹. Both have the same speed (3 ms⁻¹) but very different velocities!

Part B — True or False

FALSE
Statement 7. Speed and velocity do NOT always have the same numerical value. They are equal only when the path is a straight line in one direction. When the object changes direction, distance ≠ |displacement|, so speed ≠ |velocity|. Example: out-and-back journey — speed = 3 ms⁻¹, velocity = 0 ms⁻¹.
TRUE
Statement 8. True. If the object returns to its starting position, displacement = x_f − x_i = 0. Therefore velocity = s/t = 0/t = 0 ms⁻¹. (Average speed for the same journey would be non-zero, since distance > 0.)
FALSE
Statement 9. False. A velocity of −4 ms⁻¹ means the object is moving in the negative direction (e.g. leftward). It says nothing about whether the object is slowing down or speeding up. The magnitude of velocity (speed) is 4 ms⁻¹ — the object could be moving at constant speed in the negative direction.
TRUE
Statement 10. True. Speed is a scalar because it only has magnitude (how fast) with no directional information. Velocity is a vector because it includes both magnitude (how fast) and direction (which way). Speed = |velocity| always.

Part C — Challenge Problems

Problem 11 — Two-stage athlete
(a) 6 ms⁻¹  (b) −4 ms⁻¹  (c) 5.33 ms⁻¹  (d) 2.67 ms⁻¹ east
(a) Stage 1: Speed = 60/10 = 6 ms⁻¹ east.
(b) Stage 2: s = −20m (westward = negative), t = 5s. V = −20/5 = −4 ms⁻¹ (i.e. 4 ms⁻¹ west).
(c) Total distance = 60+20 = 80m. Total time = 10+5 = 15s. Average speed = 80/15 ≈ 5.33 ms⁻¹.
(d) Total displacement = 60−20 = 40m east. Average velocity = 40/15 ≈ 2.67 ms⁻¹ east.
Problem 12. Yes, this IS possible — as an average over a complete journey. Consider an object that walks 10m right in 1s, then 10m left in 1s (returning to start). Total time = 2s. Distance = 20m, so Speed = 20/2 = 10 ms⁻¹. Displacement = 0m, so Velocity = 0/2 = 0 ms⁻¹. This shows speed = 10 ms⁻¹ while velocity = 0 ms⁻¹ simultaneously. The key: speed uses distance (path length, never zero here) while velocity uses displacement (zero when start = end). However, at any single instant where the object is moving, both |instantaneous velocity| and speed will be equal. The difference arises for averages over direction-changing journeys.
