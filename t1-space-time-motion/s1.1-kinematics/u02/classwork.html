Reference

Speed = d/t — distance ÷ time, scalar (ms⁻¹)
Velocity (V) = s/t — displacement ÷ time, vector (ms⁻¹)
d = distance (total path, always ≥ 0)  |  s = displacement (start → end, with direction)
ms⁻¹ = m/s — negative velocity means motion in the negative direction

Part C — Short Answer

11. A car travels around a circular track and returns to its starting position. Its speed during the journey was 12 ms⁻¹. What was its average velocity for the whole journey? Explain your answer.
12. Two objects both have a speed of 5 ms⁻¹. Object A is moving right, Object B is moving left. Their velocities are described as +5 ms⁻¹ and −5 ms⁻¹. Explain why the velocities are different even though the speeds are the same.
13. Can an object have a speed of 0 ms⁻¹ but a non-zero velocity? Can it have a velocity of 0 ms⁻¹ but a non-zero speed? Explain each case.

Answer Key

Part A — Speed and Velocity Calculations

Problem 1 — Straight line east, 5s
Speed = 2 ms⁻¹ | V = 2 ms⁻¹ east
d = 10m, s = 10m east (straight line). Speed = 10/5 = 2 ms⁻¹. Velocity = 10/5 = 2 ms⁻¹ east. Path is straight so speed = |velocity|.
Problem 2 — 6m + 3m right, 3s
Speed = 3 ms⁻¹ | V = 3 ms⁻¹ right
d = 9m (6+3), s = 9m right (all in same direction). Speed = 9/3 = 3 ms⁻¹. Velocity = 9/3 = 3 ms⁻¹ right. Straight-line path in one direction.
Problem 3 — Out and back north, 4s
Speed = 4 ms⁻¹ | V = 0 ms⁻¹
d = 16m (8+8), s = 0m (returns to start). Speed = 16/4 = 4 ms⁻¹. Velocity = 0/4 = 0 ms⁻¹. Classic example: moving but displacement = 0.
Problem 4 — L-path 4+4+4m, 6s
Speed = 2 ms⁻¹ | V = 4/6 ≈ 0.67 ms⁻¹ right
d = 12m (4+4+4). Displacement: net right = 4+4 = 8m right, net up = 4m up → |s| = √(8²+4²) = √80 ≈ 8.94m at an angle. However for this level: s = 4m right (horizontal component if straight-line approximation used). Speed = 12/6 = 2 ms⁻¹. Velocity (horizontal net) = 8/6 ≈ 1.33 ms⁻¹. Note: accept horizontal displacement = 8m right if students track direction carefully: V = 8/6 ≈ 1.33 ms⁻¹ right.
Problem 5 — 15m leftward (negative), 3s
Speed = 5 ms⁻¹ | V = −5 ms⁻¹
d = 15m (always positive). s = −15m (leftward = negative direction). Speed = 15/3 = 5 ms⁻¹. Velocity = −15/3 = −5 ms⁻¹. Negative sign indicates direction.
Problem 6 — 20m right then 5m left, 5s
Speed = 5 ms⁻¹ | V = 3 ms⁻¹ right
d = 25m (20+5). s = 20−5 = 15m right. Speed = 25/5 = 5 ms⁻¹. Velocity = 15/5 = 3 ms⁻¹ right.

Part B — Number Line and Path Diagrams

Problem 7 — 0m to +9m, 3s
Speed = 3 ms⁻¹ | V = +3 ms⁻¹
d = 9m. s = +9m (rightward). Speed = 9/3 = 3 ms⁻¹. Velocity = +9/3 = +3 ms⁻¹. Straight-line path so speed = |velocity|.
Problem 8 — +8m to +2m, 2s
Speed = 3 ms⁻¹ | V = −3 ms⁻¹
d = 6m (|8−2|). s = 2−8 = −6m (moving left = negative). Speed = 6/2 = 3 ms⁻¹. Velocity = −6/2 = −3 ms⁻¹.
Problem 9 — 5R, 2L, 5R path, 4s
Speed = 3 ms⁻¹ | V = 2 ms⁻¹ right
d = 5+2+5 = 12m. s = 5−2+5 = 8m right. Speed = 12/4 = 3 ms⁻¹. Velocity = 8/4 = 2 ms⁻¹ right.
Problem 10 — +10m to −4m, 7s
Speed = 2 ms⁻¹ | V = −2 ms⁻¹
d = |10−(−4)| = 14m (straight line). s = −4−(+10) = −14m (moving leftward = negative). Speed = 14/7 = 2 ms⁻¹. Velocity = −14/7 = −2 ms⁻¹.

Part C — Short Answer

11. Average velocity = 0 ms⁻¹. The car returns to its starting position, so displacement = 0m for the whole journey. Velocity = s/t = 0/t = 0 ms⁻¹. Speed, however, was 12 ms⁻¹ throughout (using distance/time). This is the key difference between speed and velocity.
12. Speed is a scalar — it only has magnitude (5 ms⁻¹). Velocity is a vector — it includes direction. Object A moves in the positive direction (+5 ms⁻¹) and Object B moves in the negative direction (−5 ms⁻¹). Same speed, opposite directions, so their velocities are different (+5 vs −5).
13. (a) Speed = 0 but velocity ≠ 0: Not possible. If speed is 0, the object is not moving, so displacement = 0, and velocity = 0 as well. Speed is the magnitude of velocity — if magnitude is 0, the vector is also 0. (b) Velocity = 0 but speed ≠ 0: Not possible for instantaneous values. However, for an average over a journey, an object can return to start (avg velocity = 0) while having covered distance (avg speed > 0). This is shown in Example 3 of the lesson.
