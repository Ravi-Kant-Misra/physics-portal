✏️ Worked Example — GESE Method
Example: A car accelerates from rest at 10 ms⁻² for 6 seconds. What is its final velocity?
G U = 0 ms⁻¹  |  t = 6 s  |  a = 10 ms⁻²
U V = ?
E a = (V − U) / t
S Step 1: Multiply both sides by t →   ta = V − U
Step 2: Add U to both sides →   ta + U = V
Step 3: Substitute →   (6 s)(10 ms⁻²) + 0 ms⁻¹ = V
V = 60 ms⁻¹
➡️ Why Acceleration is a Vector

Acceleration is a change in velocity, which is itself a vector. To show how velocity changes over time we need to use another vector to add that change to the original velocity.

Key point: The red arrow (ΔV = 2 ms⁻¹) is the change in velocity. Dividing this by time gives acceleration: a = 2 ms⁻².
⚡ Speeding Up vs. Slowing Down
Why Direction of Acceleration Matters

The direction of acceleration matters for how the velocity changes. If the acceleration points in the same direction as the velocity, the object will speed up. If the acceleration points in the opposite direction as velocity, the object will slow down.

Speeding Up
v and a in same direction
Slowing Down
v and a in opposite directions
Rule: Same direction (v and a) → speeds up.   Opposite direction (v and a) → slows down.
🎯 Common Misconception vs. Correct Understanding
Common Misconception:
"Positive acceleration means the object is getting faster. Negative acceleration means the object is getting slower."
Correct Understanding:
"If the acceleration has the same sign as velocity, the object is getting faster. If the acceleration has the opposite sign as velocity, the object is getting slower."

🔄 3 Ways an Object Can Accelerate

1. Speeding Up

Velocity magnitude increases. a and v point the same direction.

2. Slowing Down

Velocity magnitude decreases. a and v point opposite directions.

3. Changing Direction

Even at constant speed, if direction changes, Vx and Vy change — so velocity changes — so acceleration occurs.

Important: In this and future units, it is extremely important to know exactly when an object is accelerating. An object is accelerating if any of these 3 things is happening — if it is changing its velocity. If it is not doing one of these things, it is not accelerating.
