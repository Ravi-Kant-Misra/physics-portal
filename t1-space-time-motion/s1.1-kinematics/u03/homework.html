Part B ‚ÄĒ True / False Answer Key

B1: FALSE
Correction: An object with negative acceleration is getting slower only if its velocity is positive (opposite signs). If velocity is also negative, the object is actually getting faster. The sign of acceleration alone does not determine whether the object speeds up or slows down.
B2: TRUE
Acceleration is a vector quantity (it has both magnitude and direction), and its unit is ms‚ĀĽ¬≤.
B3: FALSE
Correction: A car moving at constant speed in a straight line has zero acceleration ‚ÄĒ its velocity (speed + direction) is not changing. Acceleration is only zero if none of the three conditions apply (no speeding up, slowing down, or change in direction).
B4: TRUE
This is the core sign rule from the lesson. When acceleration and velocity have the same sign, the object's speed (magnitude of velocity) is increasing.
