Kinematics · Unit 16

Motion Maps

Drawing and interpreting motion maps to represent velocity and acceleration qualitatively

📖

What Is a Motion Map?

Definition

A motion map is a diagram that shows the position of an object at equal time intervals, together with arrows that represent the velocity (and optionally acceleration) at each position. It is a visual summary of an object's entire journey.

Key Concept

Because the time interval between consecutive dots is always 1 second, the length of the arrow connecting two consecutive dot positions is equal to the displacement per second — which is the velocity magnitude. This is why arrow length directly shows speed.

The Three Building Blocks

  • 1
    Setup: Draw a number line with a filled circle at the origin (zero point) and a rightward arrow labeled "+" to define the positive direction.
  • 2
    Dots (position markers): Place one filled dot on the number line for each second of motion. The dot shows where the object was at that moment. Label each dot: t = 0s, t = 1s, t = 2s, etc.
  • 3
    Velocity arrows (black): Draw a black arrow from each dot toward the next. The arrow length equals the displacement over that 1-second interval, which equals the velocity magnitude. Longer arrow = faster; shorter arrow = slower.
📏

Drawing Rules & Diagrams

Rule 1 — Set Up the Number Line

"First, draw a zero point and an arrow to represent the positive direction."

Reference line

The blue filled circle is the origin (zero position). The blue arrow defines which direction is positive. All motion maps must start with this reference line.

Rule 2 — Dots and Velocity Arrows (Constant Velocity)

Rule (dots): "Dots show where the object was after each second passed."

Rule (arrows): "Arrows show the 'size' of the velocity. Because velocity is displacement/time and time is 1 second for each interval, an arrow that shows the length of each interval also shows the size of the velocity."

Constant velocity — moving in positive direction

The dots are evenly spaced — equal distance covered each second. The velocity arrows are all the same length, confirming constant velocity in the positive (+) direction.

Fast vs. Slow Objects

Comparing Dot Spacing and Arrow Length

"Fast objects cover more distance in the same amount of time. Their velocity vectors are larger."

FAST Object (sports car)

Wide dot spacing — large gaps between positions each second.

Velocity arrows are long.

SLOW Object

Narrow dot spacing — small gaps between positions each second.

Velocity arrows are short.

FAST — wide spacing, long arrows
SLOW — narrow spacing, short arrows

Reading the Map

Wide spacing + long arrows = fast.  |  Narrow spacing + short arrows = slow. The arrow length between two consecutive dots equals the speed (since time = 1 s).

🟢

Standing Still (Zero Velocity)

What Does a Stationary Object Look Like?

If an object is not moving, it stays at the same position every second. All dots pile on top of one another, and because there is no displacement between them, there are no velocity arrows (or arrows of zero length).

Stationary object — all dots at same position

Why No Arrows?

Arrow length = displacement per second = 0 m / 1 s = 0 m/s. A zero-length arrow is just a dot. When an object is at rest, there is nothing to draw between the dots because they all occupy the same position.

💥

Acceleration Vectors

What Are Acceleration Vectors on a Motion Map?

In addition to velocity arrows (black), we can draw red double arrows above the velocity arrows to show acceleration.

The Acceleration Rule

"We draw a double arrow above the first to show the acceleration of the object. These arrows show how much the velocity increases or decreases in one second (the definition of acceleration). With every second, the velocity arrow will get one acceleration arrow length larger or smaller."

Key points:

  • Acceleration = change in velocity per second
  • Each second, the velocity arrow grows or shrinks by exactly one acceleration-arrow-length
  • Red arrows pointing same direction as motion = positive acceleration = object is speeding up
  • Red arrows pointing opposite to motion = negative acceleration = object is slowing down

Speeding Up in the Positive Direction Positive Acceleration

Dot spacing increases over time. Velocity arrows grow longer. Red acceleration arrows point to the right (same direction as velocity).

Accelerating object (speeding up, positive direction)

Reading the Pattern

Increasing dot gaps + longer velocity arrows + red arrows pointing RIGHT = the object is speeding up in the positive direction. Acceleration is positive.

Slowing Down (Decelerating) Negative Acceleration

Dot spacing decreases over time. Velocity arrows get shorter. Red acceleration arrows point to the left (opposite to the direction of motion).

Decelerating object (slowing down, still moving right)

Contrast With Speeding Up

Decreasing dot gaps + shorter velocity arrows + red arrows pointing LEFT (against motion) = the object is decelerating. Acceleration is negative (opposite to direction of movement).

Key Definitions

Core Concepts — Unit 16

Motion Map
A diagram showing the position of an object at equal time intervals (each 1 second), with velocity arrows indicating the magnitude and direction of velocity at each position.
Velocity on a Motion Map
Since v = displacement / time and the time interval = 1 s, the arrow length between consecutive dots equals the velocity magnitude directly.
Acceleration on a Motion Map
Shown as a red double arrow above the velocity arrow. It represents how much the velocity changes each second. Pointing with motion = speeding up; pointing against motion = slowing down.
Time Interval = 1 Second
All standard motion maps use 1-second intervals. This is the key reason that arrow length equals velocity magnitude without needing to divide by time.
