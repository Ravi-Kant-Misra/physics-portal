Answer Key — Homework Unit 14

Part A — Full Working Problems

A1. ux=8 m/s, uy=0 (horizontal launch), sy=45 m (+down), ay=9.8 m/s²
(a) Y-table, find t: s = ½at² → 45 = 4.9t² → t² = 9.184 → t = 3.03 s
(b) X-table: sx = ux×t = 8 × 3.03 = 24.2 m from the base of the cliff.
A2. u=18 m/s, θ=35°. ux=18cos35°=14.74 m/s, uy=18sin35°=10.32 m/s, ay=−9.8 m/s², sy=0 (lands at same height)
(a) Y-table, find t: 0 = 10.32t − 4.9t² → t = 10.32/4.9 = 2.106 s
(b) X-table: sx = 14.74 × 2.106 = 31.0 m
A3. u=25 m/s, θ=50°. uy=25sin50°=19.15 m/s, vy=0 at max height, ay=−9.8 m/s²
v² = u² + 2as → 0 = 19.15² − 2×9.8×sy
sy = 366.7/(2×9.8) = 366.7/19.6 = 18.7 m
A4. ux=6 m/s, uy=0, sx=18 m
(a) X-table, find t: sx = ux×t → 18 = 6t → t = 3.0 s
(b) Y-table: sy = ½×9.8×t² = 4.9×9 = 44.1 m (height of building)
A5. u=14 m/s, θ=45°. ux=14cos45°=9.90 m/s, uy=14sin45°=9.90 m/s, ay=−9.8 m/s², t=0.5 s
vx = 9.90 + 0 = 9.90 m/s
vy = 9.90 + (−9.8)×0.5 = 9.90 − 4.90 = 5.00 m/s (upward)
v = √(9.90² + 5.00²) = √(98.01 + 25) = √123.01 = 11.09 m/s
Direction: θ = arctan(5.00/9.90) ≈ 26.8° above horizontal.
A6. u=20 m/s, θ=30°. ux=20cos30°=17.32 m/s, uy=20sin30°=10 m/s, ay=−9.8 m/s², sy=+4 m
Y-table (find t): s = uyt + ½ayt²
4 = 10t − 4.9t² → 4.9t² − 10t + 4 = 0
t = [10 ± √(100 − 4×4.9×4)] / (2×4.9) = [10 ± √(100 − 78.4)] / 9.8 = [10 ± √21.6] / 9.8
t = (10 ± 4.648)/9.8 → t = 1.495 s or t = 0.546 s
Use t = 1.495 s (ball is on the way down onto the ledge, taking the later time)
X-table: sx = 17.32 × 1.495 = 25.9 m

Part B — True or False

B1. FALSE. The acceleration of a projectile is always 9.8 m/s² straight downward at every point — including at the highest point. It never changes direction. Only the velocity changes direction at the apex.
B2. FALSE. The formulas are reversed. The correct decomposition is: ux = u cos θ (horizontal) and uy = u sin θ (vertical). Cosine gives the adjacent side (horizontal); sine gives the opposite side (vertical).
B3. TRUE. "Launched horizontally" is indeed a code word meaning uy = 0 m/s. There is no initial vertical velocity when the launch direction is purely horizontal.
B4. FALSE. Time is the same in both the X-table and Y-table — it is the only variable shared between the two tables. The x and y motions happen simultaneously; the same amount of time elapses for both. This is the fundamental linking variable (Rule 3).

Part C — Challenge Problems

C1. u=15 m/s, θ=30°. ux=15cos30°=12.99 m/s, uy=15sin30°=7.5 m/s
Convention: up = +. Launch at top of building, ground is 20 m below → sy = −20 m, ay=−9.8 m/s²
(a) Y-table (find t): −20 = 7.5t − 4.9t²
4.9t² − 7.5t − 20 = 0
t = [7.5 ± √(56.25 + 4×4.9×20)] / 9.8 = [7.5 ± √(56.25 + 392)] / 9.8 = [7.5 ± √448.25] / 9.8
t = (7.5 + 21.17) / 9.8 = 28.67/9.8 = 2.93 s (take positive root)
(b) X-table: sx = 12.99 × 2.93 = 38.1 m from the base of the building.
(c) Speed just before impact:
vx = 12.99 m/s (unchanged)
vy = 7.5 + (−9.8)×2.93 = 7.5 − 28.71 = −21.21 m/s (downward)
v = √(12.99² + 21.21²) = √(168.7 + 449.9) = √618.6 = 24.9 m/s
C2. Both balls fall from 20 m with uy=0 (Ball A has horizontal speed but vertical initial velocity is still zero).
Ball A: sy=20 m, ay=9.8 m/s², uy=0 → t = √(2×20/9.8) = √4.082 = 2.02 s
Ball B: sy=20 m, ay=9.8 m/s², uy=0 → t = √(2×20/9.8) = 2.02 s
(a) They hit the ground at exactly the same time (2.02 s). The horizontal velocity of Ball A does not affect its vertical fall time — this demonstrates the independence of x and y motion.
(b) Ball B lands directly below (0 m horizontal). Ball A lands at sx = 10×2.02 = 20.2 m away.
They are 20.2 m apart horizontally.
