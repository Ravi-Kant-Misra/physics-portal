Answer Key — Classwork Unit 14

Part A — Calculations

A1. Ball off 2.0 m table at 3 m/s.
Y-table: sy=2.0 m, uy=0, ay=9.8 m/s², find t.
s = ½at² → 2.0 = 4.9t² → t² = 0.408 → t = 0.639 s
X-table: sx = ux×t = 3 × 0.639 = 1.92 m
A2. Rock kicked horizontally, t = 3 s. Cliff height = sy.
Y-table: uy=0, ay=9.8 m/s², t=3 s
sy = ½×9.8×9 = 4.9×9 = 44.1 m
A3. u=12 m/s, θ=40°. ux=12cos40°=9.19 m/s, uy=12sin40°=7.71 m/s, ay=−9.8 m/s², sy=0.
Y-table (find t): 0 = 7.71t − 4.9t² → t(7.71 − 4.9t) = 0 → t = 7.71/4.9 = 1.573 s
X-table: sx = 9.19 × 1.573 = 14.46 m ≈ 14.5 m
A4. u=20 m/s, θ=60°. uy=20sin60°=17.32 m/s, ay=−9.8 m/s², vy=0 at max height.
v² = u² + 2as → 0 = 17.32² + 2×(−9.8)×sy
sy = 300/19.6 = 15.3 m
A5. u=15 m/s, θ=25°. ux=15cos25°=13.59 m/s, uy=15sin25°=6.34 m/s. At t=1.2 s:
vx = 13.59 + 0 = 13.6 m/s (horizontal)
vy = 6.34 + (−9.8)×1.2 = 6.34 − 11.76 = −5.42 m/s (downward)
A6. Horizontal drop from 50 m, ux=40 m/s, uy=0.
Y-table: 50 = ½×9.8×t² → t² = 50/4.9 = 10.2 → t = 3.194 s
X-table: sx = 40 × 3.194 = 127.8 m ≈ 128 m

Part B — Diagram and Analysis

B1. At every point P, Q, and R: the net force direction is straight down (gravity only) and the acceleration direction is straight down at 9.8 m/s². The force and acceleration arrows should all point vertically downward at each position.
B2. vx=6 m/s (horizontal right), vy=8 m/s (downward).
Resultant: v = √(6² + 8²) = √(36+64) = √100 = 10 m/s
Direction: θ = arctan(8/6) = arctan(1.333) ≈ 53.1° below horizontal.
B3. The flat horizontal vx-time line shows constant horizontal velocity — no acceleration in x.
The vy-time graph would be a straight line with negative slope (−9.8 m/s²), starting positive (upward) and decreasing through zero (at apex) to negative (downward).
B4. (i) vy = 0 at the apex (highest point).
(ii) vy is maximum downward at the landing point (or at any point after the apex — maximum at the end).
(iii) vx is the same everywhere — it never changes. (All three positions have equal vx.)

Part C — Short Answer

C1. Newton's First Law: an object with no net force acting on it has no change in velocity. Because there is no horizontal force on the projectile (gravity acts only vertically), there is zero horizontal acceleration (ax = 0). Therefore the horizontal velocity remains constant throughout the flight.
C2. Mistake: the student mixed x-variables (sx) with y-variables (uy, ay) in a single equation. Rule 2 states you must never mix x and y variables.
Correct form for x: sx = ux×t + ½×ax×t² (and ax=0, so sx = ux×t).
C3. At 30°: ux=10cos30°=8.66 m/s, uy=10sin30°=5 m/s → t=1.02 s → range=8.9 m
At 60°: ux=10cos60°=5 m/s, uy=10sin60°=8.66 m/s → t=1.77 s → range=5×1.77=8.9 m
They travel the same range — complementary angles (30° and 60° add to 90°) always give equal range for the same launch speed. This is a property of projectile motion.
