Kinematics · Unit 14

2D Projectile Motion

Horizontal independence, parabolic paths, and the power of two tables

Definitions

2-Dimensional
Something happening on 2 perpendicular axes. A dimension is an axis of motion that you can move on, and that is at a 90° angle to other axes.
2D Projectile Motion
Movement of an object in both a horizontal and vertical direction where gravity is the only force present.

Forces on a Projectile

Key Idea: The only force acting on a projectile is gravity (Fg). Because there is no horizontal force, Newton's First Law guarantees that the horizontal velocity stays constant throughout the flight.

Acceleration of a Projectile

Because the only force acting on a projectile is gravity, the acceleration is always 9.81 m/s² straight down — at every single point along the trajectory, with the same magnitude and direction.


ax = 0 m/s²      ay = 9.81 m/s² (downward)

Velocity of a Projectile

The x-velocity of a projectile remains the same throughout the entire flight, because there is nothing to accelerate it in the x direction (ax = 0).


In the y direction, the velocity changes by 9.81 m/s downward every second due to gravity.

Tip: The total (resultant) velocity at any point is the vector sum of vx and vy. Use Pythagoras: vtotal = √(vx² + vy²). The direction changes continuously but the x-component never changes.

Fundamental Principle

Fundamental Idea: Kinematic variables on one axis are independent of kinematic variables on the other axis — except for time. We can use a separate set of kinematic variables for each axis to predict the object's motion on that axis.

The Two Kinematic Tables

Begin every 2D projectile motion problem by filling in these two tables. Notice that time does not have a direction — it is the only variable that is the same in both tables.

X-Table (horizontal)
sxuxvxaxt
Y-Table (vertical)
syuyvyayt

Three Rules for All 2D Problems

Rule 1: Because gravity is the only force on the object, the x-direction acceleration will always be 0 m/s² and the y-direction acceleration will always be 9.8 m/s² down.
Rule 2: You can only use kinematics equations on one table at a time. At NO point can you put x-axis variables and y-axis variables together into a single kinematic equation.
Rule 3: Time is the only variable that is the same in both tables. If you don't have enough variables on one table, solve for time in the other table and bring it over.

Kinematic Equations

v = u + at
Velocity–time
s = ut + ½at²
Displacement–time
v² = u² + 2as
Velocity–displacement
s = ½(u+v)t
Average velocity
Remember: These equations are used on ONE table at a time. Pick the equation that matches the variables you know and the variable you want to find.

Projectile Code Words

Phrase in the ProblemWhat It Means
"Launched horizontally" / "Falls off a flat surface" uy = 0 m/s (no initial vertical velocity)
"Highest point" / "Maximum height" vy = 0 m/s at that point
"Range" sx = total horizontal displacement
Lands at the same height it started sy = 0 m

Component Decomposition (Angled Launch)

When a projectile is launched at speed u at angle θ above the horizontal, split the velocity into components before filling in the tables:


ux = u cos θ
Horizontal component
uy = u sin θ
Vertical component

Worked Examples

Example 1 — Horizontal Launch from a Table

Problem: A ball rolls off a 1.5 m high table at 2 m/s. How far away from the table will it land?

Sign Convention

+ direction = right (x)  |  + direction = down (y)

Fill in the Tables
X-Table
sxuxvxaxt
?2 m/s—0 m/s²—
Y-Table
syuyvyayt
1.5 m0 m/s—9.8 m/s²?

"Launched horizontally" → uy = 0 m/s (code word)

Step 1 — Y-Table: Find t

Only 3 variables known on X-table, so solve Y-table first.

s = ut + ½at²

1.5 = 0×t + 0.5 × 9.8 × t²

1.5 = 4.9 t²

t² = 0.306 s²

t = 0.56 s

Step 2 — X-Table: Find sx (transfer t = 0.56 s)

s = ut + ½at²

s = 2 × 0.56 + 0

sx = 1.12 m

Answer: The ball lands 1.12 m from the base of the table.

Example 2 — Angled Launch: Find Range

Problem: A ball is launched at 10 m/s at a 30° angle above the horizontal. How far away will it land?

Sign Convention

+ direction = right (x)  |  + direction = up (y)

Component Decomposition

ux = 10 × cos(30°) = 8.7 m/s

uy = 10 × sin(30°) = 5 m/s

"Lands at same height" → sy = 0 m (code word)

Positive = up, so ay = −9.8 m/s² (gravity acts downward)

Fill in the Tables
X-Table
sxuxvxaxt
?8.7 m/s—0 m/s²—
Y-Table
syuyvyayt
0 m5 m/s—−9.8 m/s²?
Step 1 — Y-Table: Find t

s = ut + ½at²

0 = 5t + 0.5×(−9.8)×t²

0 = 5t − 4.9t²

0 = t(5 − 4.9t)

t = 0 s (launch)  or  t = 5/4.9 = 1.02 s (landing)

We use t = 1.02 s (the landing time).

Step 2 — X-Table: Find sx (transfer t = 1.02 s)

s = ut + ½at²

s = 8.7 × 1.02 + 0

sx = 8.9 m

Answer: The ball lands 8.9 m away (the range is 8.9 m).

Example 3 — Angled Launch: Find Velocity at t = 0.3 s

Problem: The same ball (10 m/s, 30°). What is its velocity after 0.3 s?

Same Components + t = 0.3 s given
X-Table
sxuxvxaxt
—8.7 m/s?0 m/s²0.3 s
Y-Table
syuyvyayt
—5 m/s?−9.8 m/s²0.3 s
X-Component of Velocity

vx = ux + ax×t = 8.7 + 0×0.3 = 8.7 m/s

Y-Component of Velocity

vy = uy + ay×t = 5 + (−9.8)×0.3 = 5 − 2.94 = 2.1 m/s (upward)

Resultant Velocity (Pythagoras)

v = √(8.7² + 2.1²) = √(75.69 + 4.41) = √80.1 ≈ 8.9 m/s

Direction: slightly above horizontal (ball is still rising at t = 0.3 s)

Answer: Velocity = 8.9 m/s at a small angle above the horizontal.
