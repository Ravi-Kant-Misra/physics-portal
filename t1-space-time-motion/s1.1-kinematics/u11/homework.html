Answer Key

Part A — Calculations

Problem 1 — Sprinter's acceleration
a = 3 ms⁻²
v = u + at → 9 = 0 + a(3) → a = 9/3 = 3 ms⁻²
Problem 2 — Braking car
s = 62.5 m
v² = u² + 2as → 0 = 625 + 2(-5)s → 10s = 625 → s = 62.5 m
Problem 3 — Rocket acceleration
a = 30 ms⁻²
v² = u² + 2as → 250000 = 40000 + 2a(3500) → 210000 = 7000a → a = 30 ms⁻²
Problem 4 — Ball thrown down (downward +)
v = 29 ms⁻¹ downward
u = +4, a = +10, t = 2.5. v = 4 + 10(2.5) = 4 + 25 = 29 ms⁻¹ (positive = downward)
Problem 5 — Ball up ramp (upward +)
s = 3.6 m
u = +6, v = 0, a = -5. v² = u² + 2as → 0 = 36 + 2(-5)s → 10s = 36 → s = 3.6 m
Problem 6 — Bus three phases
s₁ = 135 m, s₂ = 540 m, s₃ = 90 m, Total = 765 m
P1: u=0,a=1.2,t=15 → v=18 ms⁻¹, s=½(1.2)(225)=135 m. P2: u=v=18,t=30 → s=540 m. P3: u=18,v=0,t=10 → s=(18+0)(10)/2=90 m. Total=135+540+90=765 m

Part B — True or False

Statement 1
FALSE
The kinematic equations can only be used when acceleration is constant throughout the portion of motion being analysed. If acceleration changes, the journey must be split into separate phases — one SUVAT table per constant-acceleration segment.
Statement 2
TRUE
If "right" is defined as positive, any velocity in the leftward (negative) direction must be written with a minus sign in the SUVAT table. This is essential for the algebra to give the correct result.
Statement 3
TRUE
The equation v² = u² + 2as does not contain time (t), making it ideal when time is either unknown or not required. It connects velocities, acceleration, and displacement.
Statement 4
FALSE
Displacements from each phase are added together using standard vector addition (taking signs into account). If phases occur in opposite directions, the signed displacements are added — some may cancel. You do not need all phases to be in the same direction to add them.

Part C — Challenge Problems

Challenge 1:
Phase 1 (accelerating): u = 0, a = 0.5, t = 60 → v = 0 + 0.5(60) = 30 ms⁻¹ (max speed). s₁ = ½(0.5)(60)² = 900 m.
Phase 2 (constant): u = v = 30, t = 120 → s₂ = 30 × 120 = 3600 m.
Phase 3 (braking): u = 30, v = 0, t = 40 → s₃ = (30+0)(40)/2 = 600 m. Deceleration: v = u + at → 0 = 30 + a(40) → a = −0.75 ms⁻².
Total displacement = 900 + 3600 + 600 = 5100 m
Challenge 2: Define upward = positive. u = +20, a = −10.
(a) At max height v = 0: v = u + at → 0 = 20 − 10t → t = 2 s.
(b) v² = u² + 2as → 0 = 400 + 2(−10)s → 20s = 400 → s = 20 m.
(c) At t = 3.5 s: v = 20 + (−10)(3.5) = 20 − 35 = −15 ms⁻¹.
(d) The velocity is negative (−15 ms⁻¹). Since upward was defined as positive, a negative velocity means the ball is moving downward. At 3.5 s the ball has already passed its maximum height (at 2 s) and is falling back toward the ground.
