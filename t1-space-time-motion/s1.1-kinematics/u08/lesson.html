Translating Position & Velocity Graphs

Converting between P‑T and V‑T representations of motion

What Does "Translating" a Graph Mean?

Translating a graph means representing the same exact motion on a different type of graph. We focus on two directions: P‑T → V‑T (reading slopes) and V‑T → P‑T (using slopes or accumulated areas).

Key Rules to Remember

  • The slope of a P‑T graph equals the velocity at that moment.
  • The slope of a V‑T graph equals the acceleration.
  • The area under a V‑T curve equals the displacement (change in position) for that interval.
  • PT → VT: find the slope of each linear section → plot as a horizontal line on the V‑T graph.
  • VT → PT (Method 1): use each velocity value as the slope of the corresponding P‑T segment.
  • VT → PT (Method 2): calculate the area under each V‑T segment → add it to the running position.
Part 1

Converting P‑T → V‑T

Every straight segment on a P‑T graph has a constant slope, which means constant velocity. To build the V‑T graph, simply read the slope of each section and draw a horizontal line at that velocity value for the corresponding time interval.

Position–Time Graph

Velocity–Time Graph (converted)

1
Identify each linear segment of the P‑T graph and its time interval. Each segment with a straight line has a constant slope = constant velocity.
2
Calculate the slope for each segment using v = Δp / Δt = (p₂ − p₁) / (t₂ − t₁)
Example: t=0→3s, p goes 0→3 m: v = (3−0)/(3−0) = +1 ms⁻¹
3
Draw horizontal lines on the V‑T graph at the calculated velocity for each time interval. A steeper P‑T slope = a line further from zero on the V‑T graph.
Part 2

Converting V‑T → P‑T

There are two equivalent methods for converting a V‑T graph back to a P‑T graph. Use whichever feels more natural — they produce identical results.

Each velocity value on the V‑T graph becomes the slope of the corresponding P‑T segment. Start from a known initial position (usually 0) and draw line segments with the matching steepness.

  • Positive velocity → line slopes upward on P‑T graph.
  • Zero velocity → horizontal line on P‑T graph.
  • Negative velocity → line slopes downward on P‑T graph.
  • Larger |v| → steeper slope on P‑T graph.

Calculate the area under each section of the V‑T graph. This area equals the displacement (change in position) for that interval. Add each displacement to the previous position to get the new position.

Δp = v × Δt    (for constant velocity sections)
p_new = p_old + Δp
